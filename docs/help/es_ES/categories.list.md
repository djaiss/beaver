Archiva los elementos de esta colección en una estructura navegable, como carpetas. Un elemento pertenece como mucho a una categoría, y las categorías pueden anidarse a cualquier profundidad.

Para una etiqueta que atraviese todas las colecciones en su lugar, usa un tag.
