note_title: Sin posibilidad de deshacer
note: Eliminar una etiqueta la retira de todos los elementos de la cuenta a la vez. Los elementos conservan sus datos, pero la etiqueta desaparece por completo y no se puede restaurar.
---
Etiquetas de texto libre compartidas en toda la cuenta, útiles para agrupar elementos de una forma que las categorías y los tipos no permiten, como «Firmado» o «Primera edición».

Añade tantas como quieras a un elemento. Una etiqueta se aplica en todos los sitios donde se usa, así que renombrarla o eliminarla cambia cada elemento que la lleva.
