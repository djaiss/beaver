note: Refleja el mundo físico de arriba abajo: habitaciones, luego estanterías, luego cajas. Si todo cabe en un armario, una sola ubicación es un mapa perfectamente válido; añade profundidad cuando encontrar las cosas se vuelva difícil, no antes.
---
Dónde se guardan físicamente las copias, compartido en toda la cuenta para que cada colección use el mismo mapa. Las ubicaciones pueden anidarse a cualquier profundidad.

Una cuenta recién creada empieza con cinco ubicaciones ya preparadas. Renómbralas, anida bajo ellas o elimínalas para adaptarlas a tu propio espacio.
