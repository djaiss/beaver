kicker: Importar
title: Importar un tipo de colección
doc: collectionTypes.importExport
note_title: Solo se transfiere la definición
note: La importación contiene únicamente el tipo, sus campos, grupos y colores. Nunca incluye los elementos ni sus datos.
---
Recrea un tipo de colección a partir de un archivo JSON exportado desde otro tipo, para que puedas copiar una configuración entre cuentas o tomar prestada la de otra persona.

Importar siempre crea un tipo completamente nuevo. Nunca sobrescribe uno existente, así que puedes probar un documento sin riesgo: basta con eliminar el resultado si no es lo que esperabas.
