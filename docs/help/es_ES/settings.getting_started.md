La lista de bienvenida que recibe a una cuenta nueva, señalando lo poco que conviene hacer al principio: tipos de colección, etiquetas, ubicaciones y tu primera colección.

Se comparte en toda la cuenta, no es privada para ti. Ocultarla la guarda para todos, y volver a activarla aquí la trae de vuelta para todos también.
