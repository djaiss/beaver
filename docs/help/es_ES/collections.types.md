Qué tipos de colección se aplican a esta colección. Activar un tipo permite que los elementos de esta colección usen sus campos personalizados, como un número de fascículo o una añada.

Activa tantos como quieras. Un elemento elige entonces uno de los tipos activados y rellena sus campos.
