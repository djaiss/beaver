Privada, compartida o pública: para quién es esta colección. Una colección compartida está pensada para todos los miembros de tu cuenta; una pública podrá verla cualquiera con el enlace en cuanto esté disponible.

Se define de forma individual por colección, así una misma cuenta puede combinar las tres.
