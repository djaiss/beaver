El número de elementos que completan este set, por ejemplo 10 para una tirada de diez números. Beaver muestra lo que posees frente a esa cifra y cuántas piezas te faltan todavía.

Déjalo vacío para usar el set solo como agrupación, sin seguimiento de finalización.
