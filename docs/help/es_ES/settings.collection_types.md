title: Tipos de colección

note_title: Compartido entre colecciones
note: Un tipo se comparte en toda la cuenta. Cambiar sus campos cambia el formulario de cada colección que lo usa, no solo de una.
---
Un tipo decide qué campos personalizados puede registrar un elemento, como un número de fascículo para cómics o una añada para vino. Los tipos se comparten en toda la cuenta, así que uno configurado con cuidado aquí puede reutilizarse en cualquier número de colecciones.

Una cuenta recién creada empieza con una docena de tipos ya preparados para aficiones comunes. Abre el que más se parezca a la tuya y ajústalo en lugar de partir de cero.
