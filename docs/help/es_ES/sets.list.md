Una lista finita que estás completando dentro de esta colección, como una tirada completa o una lista de deseos. Dale un número objetivo y KolleK muestra lo que posees frente a esa cifra.

Deja el objetivo vacío para usar el set solo como agrupación, sin ninguna afirmación de finalización.
