Una lista finita que estás completando dentro de esta colección, como una tirada limitada de doce números. Archivar un elemento en un set aumenta su cuenta de poseídos frente al objetivo del set.

Solo un set con un objetivo mayor que cero cuenta para las estadísticas de finalización. ¿Buscas agrupar una franquicia entre varias colecciones, sin seguimiento de finalización? Usa una serie.
