Dónde se ha guardado esta copia, y durante cuánto tiempo. Cada traslado se lee como un período: la copia llegó a un sitio y luego se fue al siguiente.

Trasladar la copia es una acción aparte, no un campo que se edita, así que el historial y la ubicación actual nunca se contradicen.
