title: Claves de API personales
doc: apiKeys.manage
note_title: Se muestra una sola vez
note: El token de una clave nueva se muestra una vez, justo después de crearla. Si lo pierdes, no podrás volver a verlo: revócala y crea una nueva en su lugar.
---
Tokens personales que permiten que un script o una app actúen como tú a través de la API de KolleK. Da a cada una una etiqueta que indique para qué es, para saber más adelante cuál es seguro revocar.

KolleK te envía un correo cada vez que se crea o elimina una clave en tu usuario, para que nada pase desapercibido.
