Etiquetas de texto libre compartidas en todas las colecciones de la cuenta. Elige una existente o escribe una nueva para crearla al momento, y añade tantas como quieras.

Usa una categoría cuando la etiqueta responda a "dónde se archiva esto". Usa un tag cuando marque más bien una cualidad o un estado, como "Firmado" o "En venta".
