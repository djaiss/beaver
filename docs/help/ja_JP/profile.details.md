氏名、ニックネーム、メールアドレス、そしてアプリを表示する言語と時刻形式です。

ニックネームを設定すると、他のメンバーに表示される際に本名の代わりに使われます。言語と時刻形式は個人設定で、自分に表示される内容だけが変わります。
