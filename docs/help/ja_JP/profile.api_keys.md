---
スクリプトやアプリがKolleK APIを通じてあなたの代わりに操作するための個人用トークンです。それぞれに用途がわかるラベルを付けておくと、後でどれを無効化してよいか判断しやすくなります。

あなたのユーザーでキーが作成または削除されるたびにKolleKからメールが届くため、気づかないうちに変更が行われることはありません。
