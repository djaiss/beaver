タイプは、コミックの号数やワインのヴィンテージなど、アイテムがどのカスタムフィールドを記録できるかを決めます。タイプはアカウント全体で共有されるため、ここで丁寧に設定したタイプはいくつのコレクションでも再利用できます。

新しいアカウントには、一般的な趣味向けにあらかじめ用意された12種類のタイプが最初から含まれています。ゼロから作るより、自分の趣味に近いものを開いて調整するほうが早く済みます。
