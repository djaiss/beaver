title: コレクションタイプをインポート
doc: collectionTypes.importExport
note_title: 移動するのは定義だけです
note: インポートに含まれるのはタイプそのもの、フィールド、グループ、色だけです。アイテムやそのデータが含まれることはありません。
---
別のタイプからエクスポートされたJSONファイルからコレクションタイプを再作成します。これにより、アカウント間で設定をコピーしたり、他の人がエクスポートした設定を借りたりできます。

インポートは常に完全に新しいタイプを作成します。既存のタイプを上書きすることはないため、ドキュメントを安全に試すことができます。想定と違う場合は、結果を削除するだけで済みます。
