L'échelle de notation utilisée par les exemplaires, de Neuf à Endommagé. Partagée sur tout le compte, donc le même ensemble d'états s'applique partout.

Un compte tout neuf démarre avec cinq états prêts à l'emploi. Renommez les, réordonnez les ou supprimez les librement pour correspondre à la façon dont vous notez réellement vos objets.
