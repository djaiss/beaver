Des étiquettes libres partagées sur tout le compte, utiles pour recouper les objets d'une façon que les catégories et les types ne permettent pas, comme « Signé » ou « Première édition ».

Attachez-en autant que vous voulez à un objet. Une étiquette s'applique partout où elle est utilisée, donc la renommer ou la supprimer change chaque objet qui la porte.
