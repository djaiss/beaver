id: settings.import_collection_type
kicker: Import
title: Importer un type de collection
doc: collectionTypes.importExport
note_title: Seule la définition est transférée
note: L'import contient uniquement le type, ses champs, ses groupes et ses couleurs. Il n'inclut jamais les objets ni leurs données.
---
Recrée un type de collection à partir d'un fichier JSON exporté depuis un autre type, afin de copier une configuration entre comptes ou d'emprunter celle de quelqu'un d'autre.

Importer crée toujours un tout nouveau type. Cela n'écrase jamais un type existant, vous pouvez donc essayer un document sans risque : supprimez simplement le résultat s'il ne correspond pas à ce que vous attendiez.
