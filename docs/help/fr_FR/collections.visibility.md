Privée, partagée ou publique : à qui cette collection est destinée. Une collection partagée est destinée à chaque membre de votre compte ; une collection publique sera visible par quiconque possède le lien une fois cette fonctionnalité disponible.

Réglez le individuellement pour chaque collection, afin qu'un même compte puisse mélanger les trois.
