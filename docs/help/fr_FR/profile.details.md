Votre nom, votre surnom, votre email, ainsi que la langue et le format d'heure dans lesquels vous voyez l'application.

Un surnom, s'il est défini, remplace votre vrai nom partout où les autres membres vous voient. La langue et le format d'heure sont personnels, ils ne changent que ce que vous voyez.
