note: Le jeton d'une nouvelle clé n'est affiché qu'une fois, juste après sa création. Si vous le perdez, vous ne pourrez plus le revoir : révoquez la clé et créez en une nouvelle à la place.
---
Des jetons personnels qui permettent à un script ou une application d'agir en votre nom via l'API de KolleK. Donnez à chacune un libellé qui indique son usage, pour savoir plus tard laquelle il est sûr de révoquer.

KolleK vous envoie un e-mail chaque fois qu'une clé est créée ou supprimée sur votre utilisateur, afin que rien ne passe inaperçu.
