Une liste finie que vous cherchez à compléter à l'intérieur de cette collection, comme une série limitée de douze numéros. Classer un objet dans un set fait avancer son compte possédé par rapport à la cible du set.

Seul un set avec une cible supérieure à zéro compte dans les statistiques de complétion. Vous cherchez plutôt à regrouper une franchise entre plusieurs collections, sans suivi de complétion ? Utilisez une série.
