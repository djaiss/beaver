Un historique personnel de tout ce que vous avez fait dans ce compte, de la création d'un objet à la modification d'un paramètre.

Utilisez le pour retracer vos actions : si quelque chose semble différent de ce dont vous vous souvenez, votre journal montre si vous l'avez changé, et quand.
