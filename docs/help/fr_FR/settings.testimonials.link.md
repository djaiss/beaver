Une adresse web facultative, comme votre propre site ou profil. Si vous en ajoutez une, votre nom devient un lien cliquable à côté de vos mots sur le site marketing.

Laissez-la vide et votre nom apparaît simplement en texte brut. Dans les deux cas, vos mots s'affichent de la même manière.
