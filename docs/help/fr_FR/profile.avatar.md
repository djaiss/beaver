L'image qui vous représente dans toute l'application : dans la liste des membres, à côté de vos entrées dans le fil d'activité, et dans le menu utilisateur.

Importez un JPEG, PNG ou WebP jusqu'à 5 Mo. Il est affiché en carré, donc une image à peu près carrée rend le mieux. Sans avatar, vos initiales sont affichées à la place.
