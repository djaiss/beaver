note_title: Enregistrez vos codes de récupération
note: Perdre votre application d'authentification sans codes de récupération enregistrés peut vous verrouiller hors de votre utilisateur. Gardez les codes en lieu sûr, ailleurs que sur votre téléphone.
---
Ajoute une étape supplémentaire à la connexion : après votre mot de passe, KolleK demande un code à six chiffres depuis une application d'authentification, afin qu'un mot de passe volé seul ne suffise pas.

La désactiver retire l'étape du code et supprime vos codes de récupération ainsi que l'association avec votre application.
