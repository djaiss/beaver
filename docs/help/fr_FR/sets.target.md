Le nombre d'objets qui rendent ce set complet, par exemple 10 pour une série de dix numéros. Beaver montre ce que vous possédez par rapport à ce nombre et combien de pièces manquent encore.

Laissez le vide pour n'utiliser le set que pour regrouper, sans suivi de complétion.
