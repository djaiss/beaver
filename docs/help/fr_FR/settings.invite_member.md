Fait entrer une nouvelle personne dans ce compte, avec le rôle que vous choisissez ici : lecteur, éditeur ou propriétaire.

Seul un propriétaire peut envoyer des invitations. Commencez avec le rôle le moins élevé qui convient ; vous pourrez toujours l'augmenter ensuite.
