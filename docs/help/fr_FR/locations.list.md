Où les exemplaires sont physiquement rangés, partagé sur tout le compte afin que chaque collection utilise la même carte. Les emplacements peuvent s'imbriquer à n'importe quelle profondeur.

Un compte tout neuf démarre avec cinq emplacements prêts à l'emploi. Renommez les, imbriquez en dessous, ou supprimez les pour correspondre à votre propre espace.
