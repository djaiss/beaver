title: Nom du compte
doc: accounts.settings
---
Le nom de cet espace de travail. Il identifie le compte auprès de toutes les personnes avec qui vous le partagez, et les membres le voient sur chaque collection partagée.

Le renommer est sans risque et prend effet immédiatement. Cela ne modifie aucune collection, aucun objet ni aucun lien à l'intérieur du compte.
