note: Un type est partagé à l'échelle du compte. Modifier ses champs change le formulaire pour chaque collection qui l'utilise, pas seulement une seule.
---
Un type décide quels champs personnalisés un objet peut enregistrer, comme un numéro de fascicule pour des bandes dessinées ou un millésime pour du vin. Les types sont partagés à l'échelle du compte, donc un type configuré soigneusement ici peut être réutilisé par n'importe quel nombre de collections.

Un compte tout juste créé commence avec une douzaine de types prêts à l'emploi pour les passions courantes. Ouvrez celui qui se rapproche le plus de la vôtre et ajustez le plutôt que de repartir de zéro.
