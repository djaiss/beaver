title: Devise de valorisation
doc: accounts.settings
note_title: Pas une conversion
note: Ce changement ne fait que réétiqueter les nouveaux totaux de cette collection. Il ne convertit pas les montants déjà enregistrés, et n'affecte pas les autres collections.
---
La devise utilisée pour les totaux de valorisation de cette collection en particulier, remplaçant la devise par défaut du compte.

Utile quand les objets d'une collection sont naturellement chiffrés dans une devise différente du reste de votre compte.
