Quel type de collection est cet objet. En choisir un révèle les champs personnalisés de ce type plus bas dans le formulaire, comme un numéro de fascicule ou un millésime.

Optionnel, et peut être défini ou changé plus tard. Le laisser vide signifie simplement qu'il n'y a pas encore de champs personnalisés à remplir.
