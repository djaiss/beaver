---
id: settings.import_collection_type
kicker: Import
title: Import a collection type
doc: collectionTypes.importExport
note_title: Only the definition moves
note: The import contains just the type, its fields, groups, and colors. It never includes items or their data.
---
Recreates a collection type from a JSON file exported from another type, so you can copy a setup between accounts or borrow one from someone else's export.

Importing always creates a brand new type. It never overwrites an existing one, so it is safe to try a document and delete the result if it is not what you expected.
