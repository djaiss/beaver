title: Personal API Keys
doc: apiKeys.manage
note_title: Shown only once
note: A new key's token is shown once, right after creation. If you lose it, you cannot view it again, revoke it and create a new one instead.
---
Personal tokens that let a script or app act as you through the KolleK API. Give each one a label that says what it is for, so you know what is safe to revoke later.

KolleK emails you whenever a key is created or deleted on your user, so nothing happens unnoticed.
