---
A type decides which custom fields an item can record, such as an issue number for comics or a vintage for wine. Types are account wide, so one set up carefully here can be reused by any number of collections.

A fresh account starts with a dozen ready made types for common hobbies. Open the one closest to yours and adjust it rather than starting from scratch.
