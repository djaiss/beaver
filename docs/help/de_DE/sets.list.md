Eine endliche Liste, die du innerhalb dieser Sammlung zu vervollständigen versuchst, etwa eine vollständige Serie oder eine Wunschliste. Gib ihr eine Zielzahl, und Beaver zeigt, was du gegenüber diesem Ziel besitzt.

Lass das Ziel leer, um das Set nur zum Gruppieren zu nutzen, ohne Vervollständigungsanspruch.
