note: Das Token eines neuen Schlüssels wird nur einmal angezeigt, direkt nach der Erstellung. Verlierst du es, kannst du es nicht erneut ansehen: Widerrufe den Schlüssel stattdessen und erstelle einen neuen.
---
Persönliche Tokens, mit denen ein Skript oder eine App über die KolleK-API in deinem Namen handeln kann. Gib jedem einen Titel, der seinen Zweck nennt, damit du später weißt, welchen du gefahrlos widerrufen kannst.

KolleK schickt dir eine E-Mail, sobald an deinem Benutzer ein Schlüssel erstellt oder gelöscht wird, damit nichts unbemerkt bleibt.
