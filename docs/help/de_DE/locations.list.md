note_title: Bilde deinen echten Stauraum ab
note: Bilde die physische Welt von oben nach unten ab: Räume, dann Regale, dann Kisten. Wenn alles in einen Schrank passt, ist ein einziger Standort eine völlig gute Karte; füge Tiefe hinzu, wenn das Wiederfinden schwierig wird, nicht vorher.
---
Wo Kopien physisch aufbewahrt werden, geteilt im gesamten Konto, sodass jede Sammlung dieselbe Karte nutzt. Standorte können beliebig tief verschachtelt werden.

Ein frisch angelegtes Konto startet mit fünf fertigen Standorten. Benenne sie um, verschachtle darunter oder lösche sie, damit sie zu deinem eigenen Raum passen.
