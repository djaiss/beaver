note_title: Zwei vom selben Ding?
note: Füge diesem selben Objekt eine zweite Kopie hinzu, niemals ein zweites Objekt. Jede Kopie behält ihren eigenen Zustand, Standort und Verlauf.
---
Eine Kopie ist dein Nachweis eines physischen Exemplars, das du tatsächlich besitzt: mit eigenem Zustand, Standort, Status und Verlauf. Ein Objekt ohne Kopie ist nur eine Beschreibung.

Der Standort einer bestehenden Kopie lässt sich hier nicht bearbeiten, verschiebe sie stattdessen über ihren Verlauf, damit die Änderung dokumentiert bleibt. Der geschätzte Wert wird als eigene Schätzung gespeichert, nicht als einfaches Feld.
