Frei formulierte Labels, geteilt über alle Sammlungen im Konto. Wähle ein bestehendes aus oder tippe ein neues, um es sofort zu erstellen, und füge so viele hinzu, wie du willst.

Nutze eine Kategorie, wenn das Label beantwortet, wo etwas abgelegt wird. Nutze einen Tag, wenn es stattdessen eine Eigenschaft oder einen Zustand markiert, etwa „Signiert" oder „Zu verkaufen".
