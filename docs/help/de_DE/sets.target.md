Die Anzahl der Objekte, die dieses Set vollständig machen, etwa 10 für eine zehnteilige Serie. Beaver zeigt, was du gegenüber dieser Zahl besitzt und wie viele Teile noch fehlen.

Lass es leer, um das Set nur zum Gruppieren zu nutzen, ohne Vervollständigungsverfolgung.
