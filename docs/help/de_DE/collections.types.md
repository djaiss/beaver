Welche Sammlungstypen für diese Sammlung gelten. Einen Typ zu aktivieren erlaubt Objekten in dieser Sammlung, seine benutzerdefinierten Felder zu nutzen, etwa eine Heftnummer oder einen Jahrgang.

Aktiviere so viele du möchtest. Ein Objekt wählt dann einen der aktivierten Typen und füllt dessen Felder aus.
