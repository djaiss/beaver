kicker: Import
title: Einen Sammlungstyp importieren
doc: collectionTypes.importExport
note_title: Nur die Definition wird übertragen
note: Der Import enthält nur den Typ, seine Felder, Gruppen und Farben. Objekte oder ihre Daten sind nie enthalten.
---
Erstellt einen Sammlungstyp aus einer JSON-Datei neu, die aus einem anderen Typ exportiert wurde, sodass du ein Setup zwischen Konten kopieren oder das eines anderen übernehmen kannst.

Der Import erstellt immer einen brandneuen Typ. Ein bestehender wird nie überschrieben, du kannst ein Dokument also gefahrlos ausprobieren: Lösche das Ergebnis einfach wieder, falls es nicht das ist, was du erwartet hast.
