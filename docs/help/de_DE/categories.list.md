note_title: Löschen nimmt den ganzen Ast mit
note: Eine Kategorie zu löschen löscht auch jede darunter verschachtelte Kategorie. Objekte werden nie gelöscht, sie werden nur unkategorisiert.
---
Ordnet Objekte innerhalb dieser Sammlung in eine durchsuchbare Struktur ein, wie Ordner. Ein Objekt gehört zu höchstens einer Kategorie, und Kategorien können beliebig tief verschachtelt werden.

Für ein Label, das stattdessen jede Sammlung durchzieht, nutze einen Tag.
