doc: loans.lendAndBorrow

Besitz, der über die Zeit weg- und zurückwandert, ohne jeden Eigentumswechsel. Eine aktive Leihgabe bedeutet, dass die Kopie gerade nicht bei dir ist, egal ob du sie verliehen oder ausgeliehen hast.

Eine Leihgabe abzuschließen ist eine eigene Aktion statt einer Bearbeitung, eine offene Leihgabe trägt daher einen eigenen Schritt „Als zurückgegeben markieren".
