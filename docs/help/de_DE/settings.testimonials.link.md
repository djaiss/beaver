Eine optionale Webadresse, etwa deine eigene Website oder dein Profil. Wenn du eine hinzufügst, wird dein Name auf der Marketing-Website zu einem anklickbaren Link neben deinen Worten.

Lässt du sie leer, erscheint dein Name einfach als reiner Text. So oder so werden deine Worte gleich angezeigt.
