title: Übergeordnete Kategorie
doc: categories.organizeItems
---
Verschachtelt diese Kategorie unter einer anderen, beliebig tief. Lass sie auf der obersten Ebene, damit sie eigenständig bleibt.

Eine Kategorie unter ein neues Elternelement zu verschieben ändert nur ihren Platz im Baum. Die bereits darin abgelegten Objekte bleiben genau dort, wo sie sind.
