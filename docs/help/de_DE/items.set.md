Eine endliche Liste, die du innerhalb dieser Sammlung zu vervollständigen versuchst, etwa eine zwölfteilige Serie. Ein Objekt in ein Set einzuordnen erhöht seinen besessenen Stand gegenüber dem Ziel des Sets.

Nur ein Set mit einem Ziel über null zählt zu den Vervollständigungsstatistiken. Willst du stattdessen eine Franchise über mehrere Sammlungen hinweg gruppieren, ohne Vervollständigung zu verfolgen? Nutze eine Reihe.
