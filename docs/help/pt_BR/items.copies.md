note: Adicione uma segunda cópia a este mesmo item, nunca um segundo item. Cada cópia mantém sua própria condição, localização e histórico.
---
Uma cópia é o seu registro de uma instância física que você realmente possui: sua própria condição, localização, status e histórico. Um item sem cópia é só uma descrição.

A localização de uma cópia existente não pode ser editada aqui, mova-a a partir do histórico para que a mudança fique registrada. O valor estimado é salvo como uma avaliação própria, não como um campo simples.
