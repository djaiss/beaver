Privada, compartilhada ou pública: para quem esta coleção é destinada. Uma coleção compartilhada é destinada a todos os membros da sua conta; uma pública poderá ser vista por qualquer pessoa com o link assim que isso for lançado.

Definida individualmente por coleção, para que uma mesma conta possa combinar as três.
