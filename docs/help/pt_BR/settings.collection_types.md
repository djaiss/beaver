title: Tipos de coleção
doc: collectionTypes.setup
note_title: Compartilhado entre coleções
note: Um tipo vale para toda a conta. Alterar seus campos muda o formulário de cada coleção que o usa, não apenas de uma.
---
Um tipo decide quais campos personalizados um item pode registrar, como um número de edição para quadrinhos ou uma safra para vinho. Os tipos valem para toda a conta, então um configurado com cuidado aqui pode ser reutilizado por qualquer número de coleções.

Uma conta recém-criada já começa com uma dúzia de tipos prontos para hobbies comuns. Abra o que mais se aproxima do seu e ajuste-o em vez de começar do zero.
