Rótulos livres compartilhados em todas as coleções da conta. Escolha um já existente ou digite um novo para criá-lo na hora, e adicione quantos quiser.

Use uma categoria quando o rótulo responder "onde isto se arquiva". Use uma tag quando ela marcar uma qualidade ou um estado, como "Assinado" ou "À venda".
