id: settings.import_collection_type
kicker: Importar
title: Importar um tipo de coleção
doc: collectionTypes.importExport
note_title: Só a definição é transferida
note: A importação contém apenas o tipo, seus campos, grupos e cores. Nunca inclui itens ou seus dados.
---
Recria um tipo de coleção a partir de um arquivo JSON exportado de outro tipo, para que você possa copiar uma configuração entre contas ou pegar emprestada a de outra pessoa.

Importar sempre cria um tipo totalmente novo. Nunca sobrescreve um existente, então você pode testar um documento sem risco: basta excluir o resultado se não for o que esperava.
