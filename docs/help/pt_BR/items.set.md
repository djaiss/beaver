Uma lista finita que você está completando dentro desta coleção, como uma tiragem limitada de doze edições. Arquivar um item em um set aumenta a contagem de possuídos em relação à meta do set.

Só um set com meta acima de zero conta para as estatísticas de conclusão. Quer agrupar uma franquia entre várias coleções, sem acompanhar conclusão? Use uma série.
