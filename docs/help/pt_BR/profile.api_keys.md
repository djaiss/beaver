note_title: Exibido apenas uma vez
note: O token de uma chave nova é exibido uma vez, logo após a criação. Se você perdê-lo, não poderá vê-lo novamente: revogue a chave e crie uma nova.
---
Tokens pessoais que permitem que um script ou aplicativo aja como você através da API do KolleK. Dê a cada um um rótulo que diga para que serve, para saber depois qual é seguro revogar.

O KolleK envia um e-mail sempre que uma chave é criada ou excluída no seu usuário, para que nada passe despercebido.
