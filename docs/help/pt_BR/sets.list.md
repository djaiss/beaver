Uma lista finita que você está completando dentro desta coleção, como uma tiragem completa ou uma lista de desejos. Dê a ela uma meta e o KolleK mostra o que você possui em relação a esse número.

Deixe a meta em branco para usar o set apenas para agrupar, sem nenhuma alegação de conclusão.
