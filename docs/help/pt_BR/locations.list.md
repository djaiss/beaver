Onde as cópias são guardadas fisicamente, compartilhado em toda a conta para que cada coleção use o mesmo mapa. Locais podem se aninhar em qualquer profundidade.

Uma conta recém-criada já começa com cinco locais prontos. Renomeie-os, aninhe abaixo deles ou exclua-os para combinar com seu próprio espaço.
