A lista de boas-vindas que recebe uma conta nova, apontando o punhado de coisas que vale a pena fazer logo no início: tipos de coleção, tags, locais e sua primeira coleção.

Ela é compartilhada por toda a conta, não é privada para você. Ocultá-la a guarda para todos, e reativá-la aqui a traz de volta para todos também.
