# Categorías, sets y series

KolleK te ofrece tres formas de agrupar elementos, y se parecen entre sí hasta que conoces la pregunta que responde cada una. Esta página las desenreda para que uses siempre la adecuada.

## Las tres preguntas

- **¿Dónde se archiva este elemento?** Eso es una **categoría**.
- **¿Qué lista finita estoy intentando completar?** Eso es un **set**.
- **¿A qué franquicia pertenece esto, a través de todo lo que colecciono?** Eso es una **serie**.

## Categorías: archivar dentro de una colección

Una categoría archiva elementos dentro de una @doc(collections.overview, "colección"), y las categorías pueden anidarse. La colección "Mis cómics" de Emma tiene una categoría Marvel, con Spider-Man y X-Men anidados dentro. Explorar una categoría muestra solo sus elementos, con sus propios recuentos y estadísticas.

Una categoría pertenece a su colección y no puede salir de ella. Eliminar una categoría nunca elimina los elementos que contiene; simplemente quedan sin categorizar.

## Sets: una lista finita que completar

Un set es una lista de comprobación con un final. Emma quiere los primeros diez números de una colección, así que crea un set con un **recuento objetivo** de diez. A medida que añade elementos al set, KolleK muestra los que posee frente al objetivo, y la finalización se refleja en sus @doc(insights.collectionStatistics, "estadísticas de la colección").

Como las categorías, un set pertenece a una colección. Solo un set con un objetivo mayor que cero cuenta para las estadísticas de finalización; un set sin objetivo es solo una agrupación con nombre.

## Series: una franquicia a través de colecciones

Una serie une todo lo que pertenece a una franquicia, esté donde esté. Harry Potter abarca los libros de Emma, sus películas y sus LEGO, que son tres colecciones distintas. Una serie las atraviesa todas, y su página muestra los elementos enlazados agrupados por la colección en la que vive cada uno, junto con cuántas colecciones toca.

Una serie no hace seguimiento de finalización. Responde a "muéstrame todo lo relacionado con Harry Potter", no a "¿ya la he completado?".

## Comparativa

| | Categoría | Set | Serie |
|---|---|---|---|
| Alcance | Una colección | Una colección | Toda la cuenta |
| Se anida | Sí | No | No |
| Hace seguimiento de finalización | No | Sí, con un recuento objetivo | No |
| Pregunta que responde | Dónde se archiva | De qué lista forma parte | A qué franquicia pertenece |

Un elemento puede usar las tres a la vez: archivado bajo Marvel, formando parte del set "Primeros diez números", y enlazado a la serie Spider-Man.

Si buscas algo aún más ligero que esto, etiquetas transversales sin ninguna estructura, para eso están las @doc(tags.overview, "etiquetas").
