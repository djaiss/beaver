# Etiquetas

Las etiquetas son la forma más ligera de rotular elementos. Sin jerarquía, sin estructura, solo palabras que asocias y con las que filtras más tarde. Esta página explica para qué son buenas y en qué se diferencian de las demás herramientas de agrupación.

## Qué es una etiqueta

Una etiqueta es un rótulo corto y de formato libre, como "Firmado", "Primer número" o "Por asegurar". Las etiquetas se comparten entre todas las colecciones de la cuenta: crea "Firmado" una vez, y estará disponible tanto en los cómics de Emma como en los vinilos de Noah. Un elemento puede llevar tantas etiquetas como quieras.

Las etiquetas brillan para cualidades que atraviesan todo tu catálogo. "Firmado" se aplica a cómics, libros y discos. Una @doc(organizing.categoriesSetsAndSeries, "categoría") nunca podría expresar eso, porque las categorías viven dentro de una sola colección.

## Etiquetas frente a las demás agrupaciones

- Las **categorías** archivan un elemento en un solo lugar dentro de una colección, y pueden anidarse. Las etiquetas no imponen jerarquía y llegan a todas partes.
- Los **sets** hacen seguimiento de una lista finita hasta completarla. Las etiquetas no cuentan nada.
- Las **series** unen una franquicia. Las etiquetas no llevan más significado que la propia palabra.

Una buena regla: si el rótulo responde a "dónde se archiva esto", usa una categoría. Si marca una cualidad o un estado ("Firmado", "Duplicado", "En venta"), usa una etiqueta.

## De dónde salen las etiquetas

Cualquiera con acceso de editor o propietario puede crear una etiqueta al vuelo mientras edita un elemento, o elegir una de la lista existente. Los propietarios y editores también pueden depurar la lista compartida desde la configuración de la cuenta, tratado en @doc(tags.manageAccount).

## Eliminar una etiqueta

Eliminar una etiqueta la quita de todos los elementos que la llevaban. Los elementos en sí no se ven afectados.

:::note
Eliminar una etiqueta afecta a toda la cuenta. Elimina "Firmado" y desaparece de todos los elementos de todas las colecciones a la vez.
:::

- Mantén ordenada la lista compartida en @doc(tags.manageAccount).
- Compáralas con las agrupaciones estructuradas en @doc(organizing.categoriesSetsAndSeries).
