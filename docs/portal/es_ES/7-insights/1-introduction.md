# Análisis

Una vez que has catalogado lo suficiente, KolleK empieza a devolverte algo a cambio: lo que vale tu colección, cómo ha crecido y dónde se concentra el valor. Esta breve sección explica las pantallas que te lo muestran.

Lee las páginas en este orden:

1. @doc(insights.collectionStatistics). Cada número y gráfico de la pantalla de estadísticas de una colección, y de dónde sale cada cifra.
2. @doc(insights.dashboard). La pantalla de inicio tras iniciar sesión.

Las estadísticas son tan buenas como los datos que hay detrás. Si los números se ven escasos, la solución suele ser registrar @doc(copies.recordPaymentsAndValue, "valoraciones y transacciones") para tus ejemplares.
