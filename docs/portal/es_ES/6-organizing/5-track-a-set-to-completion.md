# Sigue un set hasta completarlo

Un @doc(organizing.categoriesSetsAndSeries, "set") es una lista finita que estás trabajando para completar dentro de una colección: una tirada limitada de doce números, un equipo completo de cartas de novato, una discografía. Dale a un set un número objetivo y KolleK te muestra lo que tienes frente al objetivo, así que "cuánto me falta" siempre tiene respuesta.

Necesitas el rol de editor o propietario para crear o cambiar sets.

## Crea un set

::::steps
:::step title="Abre los sets de la colección"
Abre la colección y ve a sus **Sets**.
:::

:::step title="Crea el set"
Elige **Nuevo set**, dale un nombre e introduce el **número objetivo**, la cantidad de elementos que contiene el set completo. Noah crea "Serie Blue Note 1500" con un objetivo de 100.

::screenshot{label="Formulario de nuevo set con el campo de número objetivo"}
:::

:::step title="Añádele elementos"
Al añadir o editar un elemento, elige el set en el formulario del elemento. Cada elemento que archives en el set hace subir el número que tienes.
:::
::::

## Cómo se cuenta la completitud

La página del set resume lo que tienes frente al objetivo. Tener 37 de una serie de 100 discos se lee como 37 de 100. La completitud también alimenta las @doc(insights.collectionStatistics, "estadísticas de la colección"), donde todos tus sets se combinan en una cifra de completitud general.

Dos detalles que conviene saber:

- **Solo un set con un objetivo superior a cero cuenta para las estadísticas de completitud.** Un set sin objetivo sigue agrupando elementos, simplemente no hace ninguna afirmación de completitud.
- **Un set que contiene más elementos que su objetivo cuenta como completo, no como sobrecompletado.** Los duplicados y variantes no te harán superar el 100 por ciento.

## Sets frente a series

Un set vive dentro de una colección y sigue la completitud. Una @doc(series.groupFranchise, "serie") abarca varias colecciones y no sigue nada; solo agrupa una franquicia. Si te preguntas "cuáles me faltan", quieres un set.

## Elimina un set

Eliminar un set lo envía a la @doc(dataSafety.restoreFromTrash, "papelera"), y se puede restaurar desde allí.

:::warning
Eliminar un set elimina la agrupación, y con ella su seguimiento de completitud. Los elementos del set no se eliminan y conservan todos sus datos.
:::

## A dónde ir después

- Agrupa una franquicia entre colecciones con @doc(series.groupFranchise, "series").
- Consulta cómo se resume la completitud en @doc(insights.collectionStatistics).
