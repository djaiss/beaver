# Desarrolladores y la API

KolleK incluye una API JSON completa que refleja la aplicación web: todo lo que puedes hacer en la aplicación, puedes hacerlo con un token y un cliente HTTP. Esta sección te guía para autenticarte y orientarte. La referencia completa de endpoints se genera a partir del código y siempre está actualizada, así que estas páginas deliberadamente no la repiten.

Lee las páginas en este orden:

1. @doc(api.overview). Qué cubre la API, cómo está estructurada, y dónde vive la referencia generada.
2. @doc(api.authenticate). Consigue un token y haz tu primera solicitud.
3. @doc(api.rateLimitsAndConventions). Paginación, dinero, límites de frecuencia y comportamiento ante errores.
4. @doc(webhooks.overview). Registra endpoints hoy, y entiende con exactitud qué se activa (y qué no) por ahora.

Para el detalle de cada endpoint (parámetros, formas de respuesta, ejemplos), usa la referencia generada en `/docs/api` de tu instancia.
