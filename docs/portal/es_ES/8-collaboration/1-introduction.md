# Colaboración

Una cuenta de KolleK es un espacio de trabajo compartido. Puedes quedártela solo para ti, o incorporar a tu hogar, tu club o tus socios de negocio, cada uno con el nivel de acceso adecuado. Esta sección trata cómo incorporar personas, decidir qué pueden hacer y compartir una colección con el exterior.

Lee las páginas en este orden:

1. @doc(collaboration.invitePeople). Envía una invitación y aprende qué experimenta la persona invitada.
2. @doc(collaboration.manageMembersAndRoles). Cambia lo que pueden hacer los miembros existentes, o elimínalos.
3. @doc(collaboration.rolesInPractice). Una referencia rápida de quién puede hacer qué.
4. @doc(collections.share). Haz que una colección sea visible para tu cuenta, o para cualquiera con un enlace.

Si todavía no has leído la página de conceptos sobre @doc(accounts.usersAndRoles, "cuentas, usuarios y roles"), empieza por ahí. Explica el modelo sobre el que se construyen estas páginas.
