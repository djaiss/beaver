# タグ

タグは、アイテムにラベルを付ける最も軽量な方法です。階層も構造もなく、ただ言葉を付けて、後でそれをもとに絞り込むだけです。このページでは、タグが得意とすることと、他のグループ化ツールとの違いを説明します。

## タグとは

タグは、「サイン入り」「創刊号」「要保険」のような、短い自由形式のラベルです。タグはアカウント内のすべてのコレクションで共有されます。「サイン入り」を一度作成すれば、Emmaのコミックにも、Noahのレコードにも使えます。1つのアイテムには好きなだけタグを付けられます。

タグは、カタログ全体を横断するような性質を表すのに向いています。「サイン入り」はコミックにも、本にも、レコードにも当てはまります。カテゴリは1つのコレクション内にしか存在しないため、これを表現することはできません。

## タグと他のグループ化との違い

- **カテゴリ**は、コレクション内の1つの場所にアイテムをファイリングし、入れ子にできます。タグは階層を持たず、あらゆる場所に及びます。
- **セット**は完成を目指す有限のリストを追跡します。タグは何も数えません。
- **シリーズ**はフランチャイズを結びつけます。タグはその言葉自体以上の意味を持ちません。

良い目安は次のとおりです。ラベルが「これはどこにファイリングされるか」に答えるなら、カテゴリを使います。性質や状態(「サイン入り」「重複」「販売中」)を表すなら、タグを使います。

## タグはどこから来るか

編集者またはオーナーの権限を持つ人は誰でも、アイテムを編集しながらその場でタグを作成したり、既存のリストから選んだりできます。オーナーと編集者は、アカウント設定から共有リストを管理することもでき、これは@doc(tags.manageAccount)で説明しています。

## タグの削除

タグを削除すると、それを持っていたすべてのアイテムからそのラベルが取り除かれます。アイテム自体には影響がありません。

:::note
タグの削除はアカウント全体に及びます。「サイン入り」を削除すると、すべてのコレクションのすべてのアイテムから一斉に消えます。
:::
