# チュートリアル

このドキュメントの他の場所にあるコンセプトページやハウツーページは、一つずつ物事を説明します。チュートリアルは、それらを組み合わせて、現実的で完結した一連の流れにまとめたものです。それぞれが、実際にありそうなコレクターの視点で最初から最後までをたどり、各手順が重要な理由を説明し、うまくいったときに何が表示されるべきかを示します。

自分の状況に合ったものを選んでください。

1. @doc(tutorials.catalogueFirstCollection, "最初のコレクションを最初から最後までカタログ化する")。Emmaが空のアカウントから、アイテム、写真、コピー、購入記録、査定を備えた実際のコミックコレクションを作り上げます。
2. @doc(tutorials.setupForHobby, "特定の趣味に合わせてアカウントを設定する")。Noahが大量のデータ入力を始める前に、レコードコレクションに合わせてタイプ、フィールド、保管場所、タグを調整します。
3. @doc(tutorials.trackValuableItem, "価値あるアイテムの一生を記録する")。Priyaが1本のヴィンテージ腕時計について、購入、査定、保険、書類、展示貸出、メンテナンス、来歴までを追跡します。
4. @doc(tutorials.inviteHousehold, "家族やクラブを招待して権限を設定する")。Emmaが自分のアカウントに2人を異なる権限で迎え入れ、コレクションを安全に共有します。
5. @doc(tutorials.selfHostWithDocker, "DockerでKolleKをセルフホストする")。Alexが何もない状態から稼働するインスタンスをインストールし、最初のインスタンス管理者を付与します。

KolleKを使うのが初めての場合は、最初のチュートリアルから始めてください。時間があまりない場合は、より短い@doc(gettingStarted.quickStart, "5分クイックスタート")から始めても構いません。
