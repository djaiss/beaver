# コレクションの表示方法を選ぶ

すべてのコレクションは3通りの方法で閲覧でき、KolleKはコレクションごとにどの方法を好んでいるかを記憶します。このページでは、各レイアウト、コレクションのヘッダーに表示される情報、そしてコレクション内検索が実際にどう動作するかを説明します。

## 3つのレイアウト

コレクションを開き、レイアウト切り替えを使って次を切り替えられます。

- **グリッド。** カバー写真をグリッドで表示します。カバーで見分けるような、視覚的なコレクションに最適です。これがデフォルトです。
- **リスト。** コンパクトな行表示で、画面によりたくさんのアイテムを表示できます。素早く一覧を眺めるのに向いています。
- **テーブル。** 項目ごとに列を表示します。アイテム間で詳細を比較したいときに最適です。

この選択は**ユーザーごと、コレクションごと**に記憶されます。Emmaは自分のコミックをグリッドで閲覧する一方、同じアカウントのSamは同じコレクションをテーブルで見ることができます。互いに影響することはなく、他のコレクションにも影響しません。

## ヘッダーに表示される情報

コレクションの上部には、**アイテム数**と**総額**が表示されます。総額は、コレクション内のすべてのコピーの現在価値の合計で、各コピーの現在価値は最新の @doc(copies.recordPaymentsAndValue, "評価額") から算出されます。一度も評価額を記録していないコピーはこの合計に含まれないため、合計額が低い場合は、たいてい価値がないのではなく、評価額が未記録であることが原因です。

## コレクション内で検索する

アイテム一覧の上にある検索ボックスは高速な絞り込み機能で、その仕組みを理解しておく価値があります。

入力するたびに、**現在ページに読み込まれているアイテム**を絞り込みます。コレクションページは一度に最大1000件のアイテムを読み込むため、ほとんどのコレクションでは実質的にすべてのアイテムが検索対象になります。コレクションがそれより大きい場合、検索は読み込まれた分だけを対象とします。

:::note
これは表示中の範囲を絞り込むフィルタであり、全体検索ではありません。開いているコレクションだけを対象とし、アカウント全体を横断して検索するものではありません。
:::

すべてのコレクションを横断するグローバル検索は計画されていますが、まだ利用できません。現状については @doc(troubleshooting.featureStatus, "機能ステータスページ") を参照してください。
