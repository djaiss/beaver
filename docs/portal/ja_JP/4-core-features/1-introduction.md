# 主要機能

概念はもう理解できました。次は実際に手を動かす番です。このセクションでは、カタログを作成し閲覧するための日常的な作業を扱います。コレクションの作成、アイテムとコピーの追加、写真のアップロード、タグ付け、そして誰が何を変更したかの確認です。

各ページは、対応する概念ページ @doc(coreConcepts.index) をすでに読んでいることを前提としており、内容を繰り返す代わりにそちらへのリンクを示します。

ページは次の順番で読んでください。

1. @doc(collections.manage)。他のすべてが入る器を用意します。
2. @doc(collections.chooseView)。グリッド、リスト、テーブルの切り替えと、コレクション内検索の挙動について。
3. @doc(items.addAndEdit)。中心となる作業です。カタログにアイテムを登録します。
4. @doc(copies.track)。それぞれの物理的なコピーとそのコンディション、保管場所、ステータスを記録します。
5. @doc(items.addPhotos)。カバー写真と追加の写真について。
6. @doc(items.tagAndFind)。コレクションを横断して使える柔軟なラベルです。
7. @doc(activity.feedAndAuditTrail)。誰が何を、いつ変更したかを確認します。

まだ何もカタログに登録していない場合は、@doc(gettingStarted.quickStart, "5分でわかるクイックスタート") がこれらの作業を最短で一通り試せる方法です。
