# アイテムに写真を追加する

写真があれば、カタログを一目で見分けられるようになります。このページでは、アイテムに写真をアップロードする方法、並び替え方法、カバー写真の選び方を説明します。

写真は1つのコピーではなく**アイテム**に紐付きます。写真はそのものの見た目を表すもので、どのコピーでも共通しています。特定の1つのコピーに関する書類(領収書、証明書、そのコピー固有のコンディション写真など)は、そのコピーの @doc(copies.attachDocuments, "書類") のほうに保管してください。

## 写真をアップロードする

アイテムの追加・編集中に、写真エリアで1枚以上の画像をアップロードできます。

- **フォーマット。** JPEG、PNG、WebP、GIF。
- **サイズ。** 1枚あたり最大10MB。

Emmaは自分のAmazing Spider-Man #300の写真を2枚アップロードします。表紙と裏表紙です。

## メイン写真を選ぶ

アイテムに複数の写真がある場合、そのうちの1枚が**メイン写真**となり、グリッドや一覧でカバーとして表示されます。デフォルトでは最初の写真がその役割を担いますが、いつでも別の写真に変更できます。追加の写真が表示される順番も並び替えられます。

## 写真を削除する

アイテムからいつでも写真を削除できます。

:::note
写真を削除すると、写真とそのファイルは即座に削除されます。写真はゴミ箱には入らないため、削除後に復元する方法はありません。
:::

## 写真を一括で管理する

アカウント全体のすべての写真は、写真ライブラリという1つの画面から検索・絞り込み・管理できます。一括操作ができるのもこの画面だけです。@doc(photos.library) を参照してください。
