# 貸し借りと保管

貸し借りとは、所有権はまったく移さずに、保管だけを一時的に移すことです。友人や画廊、美術館に作品を貸しても、それは引き続きあなたのものです。作品を借り入れても、それは引き続き他人のものです。**貸し借り**セクションは、いま自分の手を離れているもの、または借りて手元にあるもののすべてを、アカウント全体で見渡すビューです。

セクションの閲覧はどの役割にも開かれています。貸し借りの記録、返却、編集、削除には**編集者**または**オーナー**の役割が必要です。

## 二つの方向

どの貸し借りも二つの方向のいずれかを指し、セクションは一度に一つの方向を表示します。上部の切り替えを使って両者を行き来します。

- **貸出。** あなたのものでありながら、他の誰かが預かっている作品です。貸出中の外向きの貸し借りが進行中または延滞のあいだ、その複製はコレクション上で**貸出中**と表示されます。実際にはあなたの手元にないからです。
- **借入。** 他の誰かのものでありながら、いまあなたが預かっている作品です。借り入れた作品は、あなた自身の複製の表示を変えることは決してありません。もともとあなたのものではないからです。

## 各タブが示すもの

一つの方向のなかで、各タブは同じ貸し借りを別々の切り口で分けて見せます。

- **すべての貸し借り。** その方向のすべての貸し借りを、検索ボックスと、コレクション、状態、並び順のフィルターとともに表示します。
- **期限と延滞。** 三つのリストです。返却期限を過ぎた貸し借り、三十日以内に期限を迎える貸し借り、そして返却期限がまったくない無期限の貸し借りです。
- **リスクと例外。** もう一度確かめたほうがよい貸し借りです。延滞、紛失、より悪い状態で返却、返却期限なし、貸出時の状態が未記録、または書類がまったくない状態で貸し出されたもの、などです。
- **相手別。** 人や機関ごとに一枚のカードがあり、一人の借主や貸主が預かっているものすべてを一度に見渡せます。
- **保証金。** 進行中の貸し借り全体で、あなたが預かっている額や受け取るべき額と、保証金を伴う貸し借りです。
- **タイムライン。** これから迎える返却期限、最近返却された作品、最近貸し出された作品です。

上部の統計タイルはショートカットです。それぞれが、その数値に答えるタブを開きます。

## 貸し借りを記録する

先に複製を探さなくても、セクションから直接、貸し借りを始められます。

::::steps
:::step title="新しい貸し借りのドロワーを開く"
**新しい貸し借り**を選びます。方向を選んでから、コレクションから作品へ、そして動かす対象となるまさにその複製まで、一段ずつたどっていきます。
:::

:::step title="相手と日付を入力する"
作品が誰に渡るのか、あるいは誰から来るのか、手元を離れた日付、そして返却期限を入力します。返却日を取り決めていない場合は**無期限**にチェックを入れます。
:::

:::step title="状態と保証金があれば記録する"
のちの返却時に比べられるように**貸出時の状態**を選び、金銭のやり取りがあった場合は**保証金**を記録します。保証金の通貨は、既定でそのコレクションの通貨になります。
:::

:::step title="来歴の一部であれば来歴として印を付ける"
機関への貸出や展覧会の場合は**来歴に含める**にチェックを入れると、対応する来歴の出来事が生成されます。個人どうしの略式の貸し借りではチェックを外したままにすると、貸し借りの履歴だけに残ります。
:::
::::

### 複製ごとに進行中の貸し借りは一つだけ

物理的な複製は一度に一か所にしか存在できないため、一つの複製が持てる**進行中の外向きの**貸し借りは最大で一つです。すでに手元を離れている複製を貸し出そうとすると、セクションはそれを止め、まず現在の貸し借りを返却するよう求めます。この規則は JSON API でも同じく成り立ちます。

## 貸し借りを返却する

貸し借りを閉じることは編集ではなく、それ自体が一つの手順です。そうすることで、編集では捉えられないことを捉えます。

::::steps
:::step title="貸し借りを開いて返却済みにする"
どのリストからでも貸し借りを開き、**返却済みにする**を選びます。
:::

:::step title="返却を記録する"
作品が戻ってきた日付と**返却時の状態**を入力します。返却時の状態を設定すると、複製の現在の状態が更新され、複製があなたの保管のもとに戻ります。
:::
::::

返却時の状態が貸出時の状態より悪い場合、その貸し借りは損傷の可能性ありとして、貸し借りそのものの上でも、**より悪い状態で返却**のリスクリストのなかでも印が付きます。

## 手元を離れているものを書き出す

**手元を離れているものを書き出す**ボタンは、現在の方向で進行中の貸し借りを CSV としてダウンロードします。これにより、いま他人の手元にあるもの、あるいはあなたの手元にあるものを、簡素な一覧として手元に残せます。

## 関連

- 貸し借りは、複製自身の履歴にも表れます。それらが結び付く複製の記録については @doc(copies.track) を参照してください。
