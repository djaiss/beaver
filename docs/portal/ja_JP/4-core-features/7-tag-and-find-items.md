# タグでアイテムを整理・検索する

@doc(tags.overview) は、アカウント内のすべてのコレクションを横断して、アイテムに共通点をつける最も手軽な方法です。このページでは、タグの付け方、外し方、そしてタグを使いやすく保つための習慣を説明します。

## アイテムにタグを付ける

アイテムのフォームで**タグ**フィールドを使います。

- アカウントの共有一覧から**既存のタグを選ぶ**、または
- **新しいタグを入力する**と、その場で作成され、以降どこでも使い回せます。

アイテムには好きなだけタグを付けられます。Emmaはサイン入りの号には、コミックコレクションにあってもグラフィックノベルコレクションにあっても「Signed」というタグを付けます。どちらでも同じラベルです。

## アイテムからタグを外す

アイテムを編集し、タグフィールドからそのタグを取り除きます。タグ自体はアカウントの一覧に残り、それを持つ他のアイテムにも影響しません。

## 共有一覧を整理する

タグはアカウント全体で共有されるため、一覧そのものはアカウント設定という1か所で管理します。タグの名前変更や削除については @doc(tags.manageAccount) で説明しています。

:::note
アカウントからタグを削除すると、そのラベルを持っていたすべてのアイテムから取り除かれます。アイテム自体には他に一切影響しません。
:::

## タグを使いやすく保つ習慣

- **カテゴリではなく性質にタグを付ける。** 「Signed」「初刷」「要修理」「売却予定」は良いタグです。「コミック」はコレクションであり、「Marvel」はおそらく @doc(categories.organizeItems, "カテゴリ") にふさわしいものです。
- **新しく作る前に、既存のものを使い回す。** 同じ意味の表記ゆれ(「Signed」と「Autographed」)は、カタログを2つに分断してしまいます。合うタグがあれば、既存のものを選びましょう。
- **一覧を短く保つ。** 実際に絞り込みに使うタグを少数持つほうが、覚えきれない50個のタグより役立ちます。
