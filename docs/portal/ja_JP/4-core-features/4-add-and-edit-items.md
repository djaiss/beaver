# アイテムの追加と編集

これは、あなたが最も頻繁に行うことになるであろう、カタログにエントリを登録するためのページです。アイテムフォームを項目ごとに解説し、どの部分が任意なのか(ほとんどすべてです)を説明し、編集と削除についても扱います。

アイテムとコピーの違いがまだはっきりしない場合は、まず @doc(items.itemsVsCopies) を読んでください。簡単に言えば、アイテムはものの種類を説明し、コピーは実際に所有しているものを記録します。

## 誰が操作できるか

アイテムの追加と編集には **編集者** または **オーナー** の @doc(accounts.usersAndRoles, "権限") が必要です。

## アイテムを追加する

::::steps
:::step title="コレクションを開く"
アイテムが属するコレクションを開き、**新しいアイテム**を選びます。

::screenshot{label="コレクション表示の新しいアイテムボタン"}
:::

:::step title="名前を付ける"
**名前**を入力します。これが唯一の必須項目です。Emmaは「Amazing Spider-Man #300」と入力します。それ以外はすべて、今すぐでもあとからでも追加できます。
:::

:::step title="分類する"
任意で**タイプ**、**カテゴリ**、**セット**、**シリーズ**を選び、**タグ**を追加します。特に重要なのはタイプで、これを選ぶとそのタイプのカスタムフィールドがフォームに表示されます。

::screenshot{label="アイテムフォームのタイプと分類フィールド"}
:::

:::step title="詳細を入力する"
タイプが提供する**カスタムフィールド**を入力し、**写真**をアップロードし、所有している**コピー**を記録します。これらはすべて同じフォーム上で行えます。
:::

:::step title="保存する"
アイテムを保存します。すぐにコレクションに表示されます。
:::
::::

## フォームの各項目

- **名前。** 必須で、必須なのはこれだけです。
- **説明。** 他の項目に当てはまらない内容を自由に記入できます。
- **タイプ。** このアイテムがどの @doc(collectionTypes.overview, "コレクションタイプ") かを示します。コレクションで有効化されているタイプのみが選択肢に表示されます。タイプによって、下に表示されるカスタムフィールドが決まります。
- **カテゴリ。** このコレクション内でアイテムがどこに分類されるかです。@doc(categories.organizeItems) を参照してください。
- **セット。** コンプリートを目指している有限のリストです。@doc(sets.trackCompletion) を参照してください。
- **シリーズ。** 複数のコレクションにまたがりうるフランチャイズです。@doc(series.groupFranchise) を参照してください。
- **タグ。** 既存の @doc(tags.overview, "タグ") を選ぶか、新しいタグを入力するとその場で作成されます。
- **カスタムフィールド。** 選択したタイプが定義する、テキスト、数値、日付、はい/いいえのスイッチ、選択式リスト、5つ星までの評価などです。フィールドは、そのタイプで設定されたグループどおりに表示されます。
- **写真。** @doc(items.addPhotos) で詳しく扱っています。
- **コピー。** 1つ以上の物理的なコピーを、フォーム内でそのまま追加できます。@doc(copies.track) で詳しく扱っています。

一度にすべて入力しなければならないわけではありません。まず名前だけ入力し、詳細はあとで、という進め方でまったく問題なく、同じフォームがどちらにも対応します。

## アイテムを編集する

アイテムを開き、編集を選びます。同じフォームに、すでに入力された内容が表示されます。必要な部分を変更して保存します。

## アイテムを削除する

アイテムを開き、削除を選んで確定します。

:::warning
アイテムを削除すると、そのアイテムとコピーがゴミ箱に送られます。保持期間(デフォルトで30日間)が過ぎると完全に削除されます。
:::

それまでは元に戻すことができます。@doc(dataSafety.restoreFromTrash) を参照してください。
