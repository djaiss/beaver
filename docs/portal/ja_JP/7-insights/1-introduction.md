# インサイト

十分な数をカタログ化すると、KolleKはその見返りを示し始めます。コレクションの価値、その成長の様子、価値がどこに集中しているか。この短いセクションでは、それらを表示する画面について説明します。

ページは次の順番で読んでください。

1. @doc(insights.collectionStatistics)。コレクションの統計画面にあるすべての数値とグラフ、そしてそれぞれの数値がどこから来ているかです。
2. @doc(insights.dashboard)。サインイン後に表示されるランディング画面です。

統計の精度は、その裏にあるデータの質次第です。数値が乏しく感じる場合は、たいていコピーの@doc(copies.recordPaymentsAndValue, "評価と取引")を記録することが解決策になります。
