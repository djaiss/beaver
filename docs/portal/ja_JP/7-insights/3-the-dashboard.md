# ダッシュボード

ダッシュボードは、サインイン後に最初に表示される画面です。アカウントに関する短い状況報告で、誰がいて、最近何が起きたかを示します。

## 表示される内容

- **あいさつ。** 名前とともに、朝、昼、夜に応じたあいさつが表示されます。
- **メンバーと招待。** アカウントに何人いるか、保留中の@doc(collaboration.invitePeople, "招待")がいくつあるかです。
- **最近のアクティビティ。** アカウント全体で行われた最新の操作を、全メンバー分、誰が何をどれくらい前に行ったかとともに表示します。これは@doc(activity.feedAndAuditTrail, "アクティビティフィードと監査ログ")の入り口です。

そこから、ナビゲーションを使えばいつでもワンクリックでコレクションに移動できます。

## 検索ボックス

ダッシュボードには、アカウント全体からアイテムを探すための検索ボックスがあります。全体検索はまだ利用できず、このボックスは将来のための仮の設置です。1つのコレクション内での検索はすでに利用でき、@doc(collections.chooseView)で説明しています。全体検索の現在の状況については@doc(troubleshooting.featureStatus, "機能の対応状況ページ")をご覧ください。

## 新規アカウントでは表示されない

コレクションのない新規アカウントは、ダッシュボードの代わりに@doc(gettingStarted.checklist, "はじめかたチェックリスト")に案内されます。最初のコレクションが作成される(あるいはオーナーがチェックリストを非表示にする)と、以降はダッシュボードがランディング画面になります。
