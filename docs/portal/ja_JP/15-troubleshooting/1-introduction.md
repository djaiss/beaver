# トラブルシューティングとFAQ

期待していた動作にならないときは、まずここから確認してください。これらのページは最初に答えを示し、詳しい説明が必要な場合は詳細なドキュメントへのリンクを案内します。

1. @doc(troubleshooting.signIn)。パスワードを忘れた場合、認証アプリを紛失した場合、リンクの期限切れ、開けない招待についてです。
2. @doc(troubleshooting.emailDelivery)。届くはずのメールが届かない理由と、送信内容を確認する方法です。
3. @doc(troubleshooting.faq)。「コピーとは何か」から「データを取り出すにはどうすればよいか」まで、誰もが尋ねる質問への短い回答です。
4. @doc(troubleshooting.featureStatus)。現在完全に利用できる機能と、まだ準備中の機能を正直に一覧にした1ページです。

特定の機能について疑問がある場合は、通常はこれらの総合的なページよりも、各セクションの索引から該当するハウツーページへ進む方が早く見つかります。
