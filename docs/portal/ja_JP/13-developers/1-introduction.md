# 開発者向け情報とAPI

KolleKには、Webアプリケーションをそのまま反映した完全なJSON APIが備わっています。アプリでできることは、トークンとHTTPクライアントさえあればすべて実行できます。このセクションでは、認証を済ませて全体像をつかめるようにします。エンドポイントごとの完全なリファレンスはコードから自動生成され常に最新の状態を保っているため、このセクションのページではあえて内容を繰り返しません。

以下の順番でページを読み進めてください。

1. @doc(api.overview)。APIがカバーする範囲、その構造、そして自動生成されたリファレンスの場所。
2. @doc(api.authenticate)。トークンを取得し、最初のリクエストを送る。
3. @doc(api.rateLimitsAndConventions)。ページネーション、金額の扱い、レート制限、エラーの挙動。
4. @doc(webhooks.overview)。今のうちにエンドポイントを登録しておく方法と、現時点で実際に発火するもの(そしてまだ発火しないもの)の正確な理解。

エンドポイントごとの詳細(パラメータ、レスポンスの形、サンプル)については、お使いのインスタンスの `/docs/api` にある自動生成リファレンスを参照してください。
