# KolleKが送信するメール

システムが送信するすべてのメールと、その送信条件、送信先をまとめました。正規のメールかどうかを見分けたり、自分でインスタンスを運用している場合に配信状況を確認したりする際に、このページを使ってください。

KolleKは、あなたに送信したすべてのメールについて、配信状況やバウンス状況を含めて@doc(activity.logAndSentEmails, "送信済みメールページ")に記録しています。まだメーラーを設定していない運用者は@doc(selfHosting.setupEmailDelivery)を読んでください。設定前のインスタンスはメールをログに記録するだけで、実際には何も送信しません。

## サインインとその維持に関するメール

| メール | 送信条件 | 送信先 |
| --- | --- | --- |
| アカウント招待 | オーナーが誰かをアカウントに招待したとき。招待リンクは7日で失効します。 | 招待先のアドレス |
| マジックリンク | 誰かがパスワード不要のサインインリンクをリクエストしたとき。リンクの有効期限は5分間です。 | アカウントのメールアドレス |
| メールアドレスの確認 | 登録したとき、またはメールアドレスを変更したとき。 | 新しいアドレス |
| パスワードリセット | 「パスワードをお忘れですか」リンクを使用したとき。リセットリンクの有効期限は60分間です。 | アカウントのメールアドレス |

## セキュリティアラート

アカウントで注目すべき出来事が起きたときに、こちらから何もしなくても届きます。予期しないメールが届いたときの対処方法は @doc(security.alertEmails) を参照してください。

| メール | 送信条件 | 送信先 |
| --- | --- | --- |
| サインイン失敗の通知 | 既存アカウントでパスワードによるサインイン試行が失敗したとき。 | アカウントのメールアドレス |
| 新規サインインの通知 | サインインが成功したとき。使用されたデバイス名を記載します。 | アカウントのメールアドレス |
| IPアドレス変化の通知 | 前回と異なるIPアドレスからサインインがあったとき。 | アカウントのメールアドレス |
| APIキー作成 | 手動でAPIキーを作成したとき。API経由のサインインで作成されたトークンではこの通知は送信されません。 | アカウントのメールアドレス |
| APIキー削除 | APIキーを削除したとき。 | アカウントのメールアドレス |

## 運用者向けの通知

これらはコレクターではなく、インスタンスに設定されている運用者のアドレスに送信されます。サーバーを運用する人が、利用者の離脱を把握できるようにするためのものです。

| メール | 送信条件 | 送信先 |
| --- | --- | --- |
| ユーザー削除 | 本人が自分のユーザーを削除したとき。入力された理由も含まれます。 | 運用者のアドレス |
| ユーザー自動削除 | 非アクティブ時の自動削除を有効にしていたユーザーが、6か月間非アクティブだったためシステムが削除したとき。 | 運用者のアドレス |
