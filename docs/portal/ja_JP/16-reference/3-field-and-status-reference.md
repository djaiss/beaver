# フィールドとステータスのリファレンス

KolleKのフォームで出会うすべての選択肢を、1か所で一覧できるようにまとめました。各グループには、それを使うガイドへのリンクがあります。用語自体の定義については @doc(reference.glossary, "用語集") を参照してください。

## コピーのステータス

記録する各コピーに設定します。@doc(copies.track) で使用します。

| ステータス | 意味 |
| --- | --- |
| 所有中 | 現在このコピーを保有しています。新規コピーの初期値です。 |
| 注文中 | 購入済みまたは予約済みで、届くのを待っています。 |
| 貸出中 | 今は他の人の手元にあります。管理は移っていますが、所有権は移っていません。 |
| 売却済み | 売却済みで、もう所有していません。 |
| 譲渡済み | 誰かにあげました。 |
| 紛失 | 見つからず、今後も見つかる見込みがありません。 |
| 盗難 | 盗まれました。 |
| 廃棄済み | 処分またはリサイクルされました。廃棄日を任意で記録できます。 |
| その他 | 上記のいずれにも当てはまらないもの。 |

:::note
所有中、注文中、貸出中は、いずれも「まだ手元にある」状態として扱われます。貸出中のコピーもあなたの所有物であり、単に別の場所にあるだけです。
:::

## 取引の種類

各取引に設定します。@doc(copies.recordPaymentsAndValue) で使用します。「取得」に分類される種類はコピーを手元に迎えるものであり、もっとも早い取得取引がそのコピーの取得日になります。

| 種類 | 意味 |
| --- | --- |
| 購入 | コピーを購入しました。取得に分類されます。 |
| 売却 | コピーを売却しました。 |
| 交換 | 何かと交換して手に入れました。取得に分類されます。 |
| 贈与を受けた | 誰かからもらいました。取得に分類されます。 |
| 贈与した | 誰かにあげました。 |
| 相続 | 相続によって手元に来ました。取得に分類されます。 |
| 返金 | 以前の取引に対して返金がありました。 |
| 手数料 | オークション手数料など、コピーにまつわる費用です。 |
| 税金 | コピーにかかった税金です。 |
| 配送料 | 単独で記録する配送にかかった費用です。 |
| その他 | 上記のいずれにも当てはまらないお金の出来事。 |

## 評価の種類と確信度

各評価に設定します。@doc(copies.recordPaymentsAndValue) で使用します。

| 評価の種類 | 意味 |
| --- | --- |
| 自己評価 | 自分自身の判断による価値です。 |
| 専門家による鑑定 | 専門家による正式な鑑定です。 |
| 市場評価 | 現在の市場や販売データから算出したものです。 |
| 保険評価額 | 保険目的で使用する価値です。 |
| オークション評価 | オークション会社による見積もりです。 |
| 自動見積もり | 価格査定サービスやツールによって算出されたものです。 |
| その他 | その他の根拠による価値。 |

| 確信度 | 意味 |
| --- | --- |
| 低 | おおよその見当です。 |
| 中 | ある程度根拠があります。 |
| 高 | 最近の専門家による鑑定など、十分な裏付けがあります。 |
| 不明 | 確信度が記録されていません。 |

## 保険のステータス

各保険記録に設定します。@doc(copies.insure) で使用します。保険記録の補償タイプは自由入力のため、決まった選択肢の一覧はありません。

| ステータス | 意味 |
| --- | --- |
| 有効 | 現在このコピーを補償している契約です。 |
| 失効 | 補償期間が終了しました。 |
| 解約 | 満了日前に契約が解約されました。 |
| 手続き中 | 補償の手配は済んでいますが、まだ発効していません。 |

## 貸借の方向とステータス

各貸借記録に設定します。@doc(loans.lendAndBorrow) で使用します。

| 方向 | 意味 |
| --- | --- |
| 貸し出し | 自分のコピーが手元を離れました。例えば友人や展覧会に貸し出した場合です。 |
| 借り入れ | 他の人のものが今あなたの手元にあります。 |

| ステータス | 意味 |
| --- | --- |
| 予定 | 合意はされていますが、まだ受け渡しされていません。 |
| 進行中 | コピーが現在貸し出し中(または借り入れ中)です。 |
| 期限超過 | 返却期日を過ぎてもまだ戻っていません。KolleKが毎日自動でこの状態を検出します。 |
| 返却済み | 貸借が終了し、コピーが戻ってきました。 |
| キャンセル | 貸借自体が実現しませんでした。 |
| 紛失 | コピーが戻ってきませんでした。 |

## 手入れの種類

各手入れ記録に設定します。@doc(copies.recordMaintenance) で使用します。

| 種類 | 意味 |
| --- | --- |
| クリーニング | 日常的な清掃です。 |
| 修理 | 損傷を修復する作業です。 |
| メンテナンス | 時計の定期整備など、周期的なお手入れです。 |
| 保存処置 | 状態を安定させ、保存するための作業です。 |
| 修復 | コピーを以前の状態に戻すための作業です。 |
| 部品交換 | パーツや部品を交換する作業です。 |
| 点検 | 手を加えない範囲での確認作業です。 |

## 来歴イベントの種類と日付の精度

各来歴イベントに設定します。@doc(copies.traceProvenance) で使用します。

| イベントの種類 | 意味 |
| --- | --- |
| 取得 | コピーがコレクションに加わりました。 |
| 売却 | コピーが売却されました。 |
| 贈与 | コピーが贈与によって渡りました。 |
| 相続 | コピーが遺産として引き継がれました。 |
| 所有権の移転 | その他の方法で所有権が変わりました。 |
| 管理の移転 | 所有者を変えずにコピーが移動しました。 |
| 貸出 | コピーが貸し出されました。 |
| 返却 | 貸し出されていたコピーが戻ってきました。 |
| 展示 | コピーが一般に公開されました。 |
| 真贋鑑定 | コピーが本物であると確認されました。 |
| 鑑定 | コピーが正式に評価されました。 |
| 大規模な修復 | 物語に残るほどの大規模な作業です。 |
| 由来 | コピーがいつ、どこで作られたかです。 |
| 発見 | コピーが発見または再発見されました。 |
| その他 | その他、物語の1章となる出来事。 |

来歴の日付はしばしば不確かなため、各イベントには精度が設定されます。

| 精度 | 意味 |
| --- | --- |
| 正確な日付 | 完全な日付が判明しています。 |
| 月 | 月まで判明しています。 |
| 年 | 年まで判明しています。 |
| おおよそ | 最善の推定値です。「およそ」として読んでください。 |
| 不明 | 日付が記録されていません。 |

## 書類の種類

各書類に設定します。@doc(copies.attachDocuments) で使用します。

| 種類 | 意味 |
| --- | --- |
| 領収書 | 購入の証明です。 |
| 請求書 | コピーやそれに関する作業の請求書です。 |
| 証明書 | コピーに付属していた証明書です。 |
| 鑑定書 | 書面による評価です。 |
| 保険関連書類 | 契約に関する書類です。 |
| 写真 | ギャラリー用ではなく、記録として保管する写真です。 |
| コンディションレポート | コンディションについての書面による評価です。 |
| 修復レポート | 修復作業の記録です。 |
| カタログ | カタログの掲載ページや目録です。 |
| 通信記録 | コピーに関する手紙やメールです。 |
| 所有権証明書類 | 所有権を証明する書類です。 |
| 真贋証明書類 | コピーが本物であることを証明する書類です。 |
| その他 | その他、保管しておく価値のあるもの。 |

## カスタムフィールドの種類

コレクションタイプ上でカスタムフィールドを定義する際に選びます。@doc(collectionTypes.setup) で使用します。

| フィールドタイプ | 意味 |
| --- | --- |
| テキスト | 著者名や出版社名など、自由形式のテキストです。 |
| 数値 | 号数など、数値による値です。 |
| 日付 | 発売日など、カレンダー上の日付です。 |
| はい/いいえ | 「サイン入り」など、チェックボックスによる項目です。 |
| 選択式 | 自分で定義した選択肢の一覧から1つを選びます。 |
| 評価 | 星の数による評価で、最大5つ星です。 |

## コレクションの公開範囲

各コレクションに設定します。@doc(collections.share) で使用します。この設定は現時点では記録のみで、共有機能が実装され次第適用されます。詳しくは @doc(troubleshooting.featureStatus) を参照してください。

| 公開範囲 | 意味 |
| --- | --- |
| プライベート | 自分だけを対象としています。 |
| 共有 | アカウント内の全員を対象としています。 |
| 公開 | サインインなしでリンクを知っている誰でも、閲覧のみできる状態を対象としています。 |
