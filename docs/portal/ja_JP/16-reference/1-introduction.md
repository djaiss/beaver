# リファレンス

このセクションのページは、通読するためではなく、必要なときに参照するための資料です。他のページで用語やオプション、メールについて触れられていて、その全体像を知りたいときは、ここに答えがあります。

1. @doc(reference.glossary)。すべての製品用語を1、2行で定義し、詳しく説明しているページへのリンクを添えたもの。
2. @doc(reference.fieldAndStatus)。コピーのステータスから来歴イベントの種類まで、フォームで出会うすべての選択肢を、1行の意味とともにまとめたもの。
3. @doc(reference.emailsSent)。システムが送信するすべてのメールと、その送信条件、送信先。

KolleKを初めて使う場合は、ここから始めないでください。まずは@doc(gettingStarted.index)から始め、何かを調べたくなったらここに戻ってきてください。
