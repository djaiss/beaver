# 推薦の声を書く

KolleKによってコレクションの管理が楽になったなら、その気持ちを推薦の声として寄せていただけます。いただいた声は、他のコレクターの目に触れるよう、公開マーケティングサイトに掲載させていただく場合があります。このページでは、推薦の声の送り方、送った後にどうなるか、そしてあなたがその声に対して保ち続けられる主導権について説明します。

この機能は、お使いのインスタンスでマーケティングサイトが有効になっている場合にのみ表示されます。**設定 → マーケティング → 推薦の声**から利用できます。アカウントのメンバーであれば、権限にかかわらず誰でも送信できます。

## 推薦の声とは

推薦の声とは、KolleKを何のために使い、どこを気に入っているかを、あなた自身の言葉でつづった短く正直なメッセージです。次の3つの要素で構成されます。

- **名前。** あなたの言葉の横に表示したい名前です。アカウント名である必要はありません。
- **リンク(任意)。** ご自身のサイトなどのウェブアドレスです。追加すると、マーケティングサイト上であなたの名前がクリックできるリンクになります。
- **言葉そのもの。** 1〜2文がもっとも読みやすいです。友人に話しかけるように書いてみてください。

もう1つのルールがあります。**英語**で書いてください。マーケティングサイトは英語のみのため、他の言語で書かれた推薦の声は掲載できません。

## 1人につき1つの推薦の声

各メンバーが保持できる推薦の声は1つだけです。もう一度送信すると、2つ目が追加されるのではなく、以前のものが置き換わります。これは修正の方法でもあります。文面を編集して、もう一度送信してください。

## 送信した後にどうなるか

送信すると、あなたの推薦の声は**審査中**になります。何かが公開される前に、インスタンスの運営者がすべての送信内容に目を通します。これは、掲載欄を本物のままに保ち、スパムを排除するためです。

- **掲載された**場合、あなたの言葉はKolleKのホームページと推薦の声のページに公開され、お礼のメールをお送りします。
- ふさわしくない場合は、単に掲載されません。修正して、もう一度送信していただけます。

審査中も、掲載された後も、推薦の声の画面には、あなたが送信した内容と、その現在の状態が正確に表示されます。

## 主導権はあなたにあります

推薦の声はいつでもあなたが取り下げられます。まだ審査中であっても、すでに公開されていても、いつでも削除でき、削除するとすぐにマーケティングサイトから外れます。推薦の声の画面にある**推薦の声を削除**ボタンを使ってください。

コレクションそのものを他の人と共有する方法については、@doc(collaboration.index)を参照してください。
