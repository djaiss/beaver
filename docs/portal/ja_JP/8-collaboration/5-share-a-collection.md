# コレクションを共有する

すべてのコレクションには、それが誰に向けたものかを記録する公開範囲の設定があります。このページでは、その設定方法を説明するとともに、この設定が現時点で実際に何をするのかを正直にお伝えします。共有はKolleKがまだ発展させている途中の機能だからです。

3段階のレベルについては、@doc(sharing.overview, "公開範囲と共有")で詳しく説明しています。

- **プライベート。** 自分だけを対象としています。
- **共有。** アカウント内の全員を対象としています。
- **公開。** リンクを知っている誰でも、サインインなしで閲覧のみできることを対象としています。

:::note
現時点では、この設定はあなたの意図を記録するだけで、まだ実際には適用されていません。アカウントのすべてのメンバーは、依然としてすべてのコレクションを閲覧できますし、配布できる公開リンクもまだ存在しないため、アカウントの外から到達できるものは何もありません。現在の状況は@doc(troubleshooting.featureStatus)で確認できます。
:::

## コレクションの公開範囲を変更する

::::steps
:::step title="コレクション設定を開く"
@doc(collections.overview, "コレクション")を開き、編集を選びます。公開範囲は、名前や絵文字と同じく、コレクション自体の設定項目にあります。
:::

:::step title="公開範囲を選ぶ"
**プライベート**、**共有**、**公開**のいずれかを選びます。

::screenshot{label="3つの公開範囲の選択肢が表示されたコレクションフォーム"}
:::

:::step title="保存する"
コレクションを保存します。設定はすぐに保存されます。
:::
::::

Emmaは「Wishlist Research」コレクションをプライベートに、「My Comics」をSamとLeoのために共有に、「Vintage Posters」を公開に設定し、いつかリンクを配れる日に備えています。

## 今のうちに設定しておく理由

今のうちに公開範囲を設定しておけば、共有機能が有効になった瞬間から、あなたのカタログは正しく振る舞います。プライベートに設定したコレクションはそのままプライベートになり、公開に設定したコレクションはそのままリンクを取得します。その時点で改めてすべてのコレクションを見直す必要はありません。

:::warning
公開リンクが実装されると、公開設定のコレクションは、そこに含まれるすべてのアイテム、写真、詳細情報を含めて、リンクを知っている誰でもサインインなしで閲覧できるようになります。それで問題がある場合、特に何を所有していて、どこに住んでいるかが分かってしまう場合は、コレクションを公開に設定しないでください。
:::

## 再びプライベートに戻す

コレクションを編集し、公開範囲をいつでも**プライベート**(または**共有**)に戻せます。設定はその時点からそのコレクションに適用されます。

:::note
共有は、誰がコレクションを見られるかを制御します。メンバーがそのコレクションで何をできるかは、公開範囲ではなく権限によって決まります。閲覧者は何も編集できず、編集者は編集できます。@doc(collaboration.rolesInPractice, "実践で見る3つの権限")を参照してください。
:::
