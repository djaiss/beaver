# コラボレーション

KolleKのアカウントは共有ワークスペースです。自分だけで使うこともできますし、家族やクラブ、ビジネスパートナーを招き入れて、それぞれに適切な権限を与えることもできます。このセクションでは、人を招待する方法、その人ができることを決める方法、そしてコレクションを外部と共有する方法を説明します。

以下の順番でページを読み進めてください。

1. @doc(collaboration.invitePeople)。招待の送り方と、招待された人が経験することを学びます。
2. @doc(collaboration.manageMembersAndRoles)。既存のメンバーができることを変更したり、メンバーを削除したりします。
3. @doc(collaboration.rolesInPractice)。誰が何をできるかを一覧で確認できる資料です。
4. @doc(collections.share)。コレクションをアカウント内に見せる、あるいはリンクを知っている誰にでも見せるようにします。

まだ@doc(accounts.usersAndRoles, "アカウント、ユーザー、権限")の概念ページを読んでいない場合は、先にそちらから読み始めてください。このセクションのページが前提としているモデルを説明しています。
