# アカウントとプロフィール

他のメンバーの管理とは別に、自分自身についても管理します。名前、言語、各種設定、そしてオーナーであればアカウント全体の設定です。このセクションでは、そうした個人向けのページをまとめています。

ページは次の順番で読んでください。

1. @doc(profile.edit)。名前、ニックネーム、メールアドレス、各種設定について。
2. @doc(profile.setAvatar)。プロフィール画像のアップロードと削除について。
3. @doc(profile.changeLanguage)。7つのインターフェース言語から選択します。
4. @doc(activity.logAndSentEmails)。自分が行った操作と、KolleKから送られたメールを確認します。
5. @doc(accounts.settings)。オーナー向けの、アカウント名、デフォルト通貨、チェックリストについて。

パスワード、2段階認証、APIキーといったセキュリティ関連の内容は、代わりに @doc(security.index, "セキュリティ") セクションを参照してください。
