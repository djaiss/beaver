# アバターを設定する

アバターは、メンバー一覧、アクティビティフィード上の自分の投稿の隣、ユーザーメニューなど、KolleK内であなたを表す小さな画像です。設定していない場合、KolleKは自動生成されたプレースホルダーを代わりに表示します。このページでは、アップロード方法と取り消し方法を説明します。

## アバターをアップロードする

@doc(profile.edit, "プロフィール") から、アバターの変更を選び、画像ファイルを選択します。

いくつかの実用的なルールがあります。

- **JPEG**、**PNG**、**WebP**のいずれかの画像を使ってください。
- ファイルサイズは最大**5MB**です。
- 画像は正方形で表示されるため、おおよそ正方形の画像が最も見栄えよく表示されます。横長や縦長の画像は切り抜かれます。

保存すると、新しいアバターがすぐにあらゆる場所に反映されます。

## アバターを削除する

同じ場所からアバターの削除を選びます。KolleKは自動生成されたプレースホルダーの表示に戻ります。いつでも新しい画像を再度アップロードできます。
