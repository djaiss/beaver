# コピーに書類を添付する

領収書、鑑定書、証明書、保険証券。よく記録されたコピーには書類が伴います。KolleKはその書類をコピー自体に、あるいはさらに良い方法として、それが属する正確な記録に保存します。領収書は取引と、証明書は真贋鑑定イベントと一緒に保管されます。

## 書類を添付できる場所

書類は**コピー自体**、またはその履歴記録のいずれか一つに添付できます。

- **取引**(領収書や請求書)
- **評価**(書面による鑑定書)
- **保険記録**(保険の明細書)
- **貸借**(署名済みの貸借契約書)
- **メンテナンス記録**(サービス伝票、作業前後の写真)
- **来歴イベント**(証明書、アーカイブの記載抜粋)

できるだけ具体的な場所に添付してください。購入取引に添付された領収書はそれだけで意味が通じますが、同じファイルをコピーに漠然と添付すると、後で見たときに推測が必要になります。特定の記録に属さない書類、例えば全体的なコンディションを示す写真一式などは、コピー自体に添付してください。

## ファイルかリンクか

書類は**アップロードしたファイル**か、**外部リンク**のいずれかです。

アップロードできるのは、PDF、一般的な画像形式(スマートフォンで撮影したHEIC写真を含む)、WordやExcelのファイル、CSV、プレーンテキストで、1ファイルあたり最大12MBです。保存されたファイルは非公開で、あなたのアカウントにサインインしたメンバーにのみ配信され、公開URLから配信されることはありません。

外部リンクは、オークションの出品ページやメーカーのアーカイブの記載など、他の場所にあるものを指します。KolleKが保存するのはページのコピーではなく、そのアドレスです。

:::note
リンク先のページは変更されたり消えたりする可能性があり、KolleKが保存するのはアドレスのみです。失っては困るものは、ダウンロードしてファイルとしてアップロードしてください。
:::

## 書類を添付する

Priyaは、1968年製オメガ スピードマスターのオークション領収書を、その購入取引に添付します。

::::steps
:::step title="コピーの履歴を開く"
アイテムを開き、**履歴**タブに移動して、コピーを選択し、**書類**セクションを開きます。

::screenshot{label="書類セクションを開いた履歴タブ"}
:::

:::step title="添付先を選ぶ"
書類を追加し、その保存先を選びます。コピー自体か、いずれかの記録です。Priyaは購入取引を選びます。
:::

:::step title="ファイルまたはリンクを入力する"
**ファイル**をアップロードするか、代わりに**外部リンク**を貼り付けます。
:::

:::step title="内容を記述する"
**書類の種類**を選び、**名前**(「オークション領収書、Fine Time Auctions」など)を入力し、必要に応じて**説明**、**発行日**、**参照番号**を入力します。保存します。
:::
::::

## 書類の種類

種類を分けることで、大量の書類も一目で見分けられるようになります。**領収書**、**請求書**、**証明書**、**鑑定書**、**保険**、**写真**、**コンディション報告書**、**修復報告書**、**カタログ**、**通信記録**、**所有記録**、**真贋記録**、**その他**です。

:::note
書類はアイテムの写真ではありません。アイテムの見た目を示す画像は、写真ギャラリーに属します(@doc(items.addPhotos)を参照)。書類として添付する写真は証拠であり、例えば貸借に紐づいた日付入りのコンディション写真などが該当します。
:::
