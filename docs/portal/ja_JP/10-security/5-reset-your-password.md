# パスワードをリセットする

パスワードを忘れた場合でも、単に新しいものに変更したい場合でも、このページでは両方の方法を説明します。サインインページからアクセスを回復する方法と、プロフィールから意図的にパスワードを変更する方法です。

## パスワードを忘れた場合

1. サインインページで**パスワードをお忘れですか**のリンクを選びます。
2. メールアドレスを入力して送信します。
3. KolleKから届いたメールを開き、リセット用のリンクをたどります。
4. 新しいパスワードを設定して確定します。これでそのパスワードでサインインできるようになります。

紛らわしくないように、知っておくべき2つの挙動があります。

- **確認メッセージは常に同じです。** 入力したアドレスにアカウントが存在するかどうかにかかわらず表示されます。これは、誰が登録済みかを決して明らかにしないことで、プライバシーを保護するためです。アカウントがある場合は、メールが届きます。
- **リセット用リンクの有効期限は60分間です。** 開くのが遅れて期限切れになった場合は、もう一度リクエストしてください。

:::note
リセットの手順自体を省略したい場合は、@doc(auth.magicLinks, "マジックリンク") を使えばパスワードなしでサインインできます。サインインしたあとは、プロフィールから新しいパスワードを設定できます。
:::

## 単に変更したい場合

パスワードを変更するのに、パスワードを忘れた場合のフローを使う必要はありません。プロフィールに移動してセキュリティのエリアを開き、そこでパスワードを変更します。現在のパスワードを入力し、新しいパスワードを選びます。

## パスワードが拒否される理由

KolleKは新しいパスワードを2つのルールに照らしてチェックするため、拒否された理由が不明なことはありません。

- **8文字以上であること。** これより短いパスワードは無条件で拒否されます。
- **漏洩が確認されているパスワードでないこと。** 入力したパスワードの候補は、公開されているデータ漏洩事件に登場したパスワードのリストと照合されます。どこかで一度でも漏洩したことがあれば、一見強力に見えても拒否されます。これはパスワードそのものに関する問題であってアカウントに関する問題ではないため、他のサイトで使ったことのないものを選んでください。

パスワードマネージャーを使えば、長くて一意なパスワードを生成できるので、どちらのルールも苦労せずクリアできます。
