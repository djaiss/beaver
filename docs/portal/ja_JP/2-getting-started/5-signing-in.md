# サインイン

KolleKにはいくつかのサインイン方法が用意されています。このページではそれぞれの方法と、ロックアウトされてしまった場合の対処先を説明します。

## メールアドレスとパスワードでサインインする

通常の方法です。サインインページに移動し、登録した**メールアドレス**と**パスワード**を入力して送信します。ダッシュボードに移動します。

アカウントで@doc(security.twoFactorAuth, "2段階認証")が有効になっている場合、パスワードの直後にコードの入力を求められます。詳しくは下記を参照してください。

## マジックリンクでサインインする

パスワードを入力したくない場合、KolleKはサインイン用のリンクをメールで送信できます。

サインインページでマジックリンクのオプションを選び、**メールアドレス**を入力して送信します。KolleKはそのアドレスに1回限り有効なリンクを送信します。リンクを開くとサインインが完了します。

知っておくべき点が2つあります。

- **リンクの有効期限は5分間です。** 期限切れの場合は、もう一度リクエストしてください。
- **リンクはアカウントのメールアドレス宛てに送信されます。** そのため、そのメールボックスにアクセスできる必要があります。これは同時に安全性の担保にもなっており、メールを読める人だけがリンクを使用できます。

## 2段階認証のステップ

2段階認証を有効にしている場合、パスワードでのサインインにもう1ステップ加わります。パスワードが承認された後、KolleKは認証アプリに表示されている現在のコードの入力を求めます。入力すればサインインが完了します。

認証アプリにアクセスできない場合は、代わりに@doc(security.recoveryCodes, "リカバリーコード")のいずれかを入力できます。各リカバリーコードは1回のみ使用できます。

:::warning
マジックリンクでのサインインでは、2段階認証のコードは求められません。メールの受信箱にアクセスできること自体が、すでに第2要素として機能しているためです。2段階認証に依存している場合は、この点を踏まえてサインイン方法を選び、メールアカウントを適切に保護してください。
:::

2段階認証の設定とリカバリーコードの保存については、このドキュメントの**セキュリティ**セクションで説明しています。

## パスワードを忘れた場合

パスワードを思い出せない場合は、サインインページの「パスワードをお忘れですか」リンクを使用してください。メールアドレスを入力すると、KolleKがリセット用リンクを送信します。

プライバシー保護のため、KolleKはそのアドレスにアカウントが存在するかどうかにかかわらず、常に同じ確認メッセージを表示します。そのため、このページから誰が登録済みかを知ることはできません。アカウントを持っている場合は、リセット用メールが届きます。マジックリンクで再度サインインした場合は、後でプロフィールからパスワードをリセットできます。
