# はじめに

KolleKへようこそ。このセクションでは、「見つけたばかり」の状態から最初のアイテムを登録するところまでを、あらかじめ難しい知識を身につけることなく案内します。

以下の順番でページを読み進めてください。

1. @doc(kollek.whatIs)。KolleKが何をするサービスで、自分に合っているかどうか。
2. @doc(kollek.hostingOptions)。どのように運用するか、そしてアプリ内で支払いが発生しない理由。
3. @doc(accounts.create)。サインアップしてワークスペースを作成する。
4. @doc(auth.signIn)。次回以降サインインするあらゆる方法。
5. @doc(gettingStarted.checklist)。最初にやっておくべき5つのこと。
6. @doc(gettingStarted.quickStart)。最初のアイテムを最初から最後まで登録する。

すでにKolleKが何であるかを知っていて、すぐに始めたい場合は、@doc(accounts.create)または@doc(gettingStarted.quickStart, "クイックスタート")に進んでください。
