# はじめのチェックリスト

新しいアカウントを初めて開くと、KolleKは空のダッシュボードの代わりに短いチェックリストを表示します。これは、その後の作業をすべて楽にしてくれる、いくつかのことを指し示すものです。このページでは、その目的と動作を説明します。

## チェックリストとは何か

これは友好的な出発点の地図であり、ルールの集まりではありません。コレクションを始めたばかりのときにやっておく価値のある、5つのことを提案します。

- **@doc(collectionTypes.overview, "コレクションタイプ")を設定する。** 何を記録し、それぞれの種類にどんな詳細を記録すべきかを決めます。コミック、レコード、LEGOなど、何でも構いません。
- **@doc(tags.overview, "タグ")を設定する。** 「サイン入り」や「初版」など、後でアイテムを横断的に絞り込むために使える、柔軟なラベルをいくつか作成します。
- **他の@doc(accounts.usersAndRoles, "メンバー")を追加する。** 自分1人だけで使っているのでなければ、アカウントを一緒に閲覧または管理する人を招待します。
- **@doc(locations.overview, "保管場所")を追加する。** 棚、箱、部屋など、ものがどこにあるかを記述します。
- **最初の@doc(collections.overview, "コレクション")を追加する。** 一番楽しい部分です。登録を始めましょう。

この順番どおりに行う必要はなく、すべてを行う必要もありません。多くのコレクターは、まずコレクションの作成に進み、残りは後で戻ってくることが多いです。このリストはあくまで提案として捉えてください。

## 自動的にチェックが付く

「完了にする」ボタンはありません。実際にその作業を行うと、各ステップは自動的に完了になります。保管場所を追加すれば保管場所のステップにチェックが付き、コレクションを作成すればそのステップにチェックが付きます。

:::note
アカウントに最初から用意されているコレクションタイプや保管場所は、チェックリストの達成にはカウントされません。ステップが完了とみなされるのは、実際に自分で何かを作成または変更したときだけなので、チェックリストは最初から用意されていたデフォルトの内容ではなく、あなた自身の設定状況を反映します。
:::

## アカウント全体で共有される

このチェックリストはアカウント全体で共有されており、個人専用のものではありません。アカウントを開く全員が同じ進捗を目にします。誰か1人のメンバーが最初のコレクションを作成すれば、そのステップは全員にとって完了として表示されます。

## チェックリストを非表示にする

軌道に乗ったら、チェックリストをしまうことができます。

これはアカウント全体に関わる画面であるため、非表示にできるのは**@doc(accounts.usersAndRoles, "オーナー")**のみです。オーナーであれば、非表示にするオプションが表示されます。非表示にすると、アカウント内の全員に対して表示されなくなります。

完全に失われるわけではありません。オーナーはいつでもアカウント設定からチェックリストを再表示できます。また、コレクションがまだない真新しいアカウントでは、登録するものができるまでチェックリストが表示され続けます。
