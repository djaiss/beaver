# KolleKとは

KolleKは、あなたが集めているものすべてのための居場所です。コミック、レコード、コイン、腕時計、ワイン、本、トレーディングカード、ビデオゲーム、あるいはあなただけが好きな妙にニッチなもの。所有できるものであれば、ここに登録できます。

## KolleKが解決する課題

多くのコレクターは、最終的にスプレッドシート、メモアプリ、写真フォルダ、そして自分の記憶に情報が散らばってしまいます。コレクションが小さいうちはそれでも何とかなりますが、増えてくると、いくら払ったか、どの棚に置いたか、今誰かに貸しているか、全体でいくらの価値があるかを思い出せなくなります。

KolleKは、これらすべてをコレクション専用に作られた1つのツールに置き換えます。

## どのように整理されているか

始めるにあたって覚えておくべき考え方は1つだけです。KolleKのすべては4つの階層に分かれています。

- **@doc(accounts.usersAndRoles, "アカウント")** はあなたのワークスペースです。作成したものはすべてこの中に存在します。
- **@doc(collections.overview, "コレクション")** は「My Comics」や「Wine Cellar」のような、名前の付いたグループです。
- **@doc(items.itemsVsCopies, "アイテム")** は「Amazing Spider-Man #1」のような、ものの種類です。
- **@doc(items.itemsVsCopies, "コピー")** は実際に所有している1つの物理的なものです。

アイテムとコピーの違いこそが、KolleKを単なるリストとは違うものにしています。同じコミックを3冊持っている場合、それは3つのコピーを持つ1つのアイテムであり、それぞれのコピーは独自のコンディション、保管場所、価値、履歴を持ちます。この考え方は実際に登録を始めるときにきちんと理解できるので、今はまだ気にする必要はありません。

:::note
アイテムとコピーの違いは、今後ずっと使い続けることになる、覚えておく価値のある唯一の考え方です。
:::

## どんな人向けか

KolleKは、非公開のカタログを管理する1人のコレクターにも、カタログを共有するグループにも使えます。アカウントに他の人を招待し、それぞれに閲覧、編集、管理のどの権限を与えるかを決めることができます。家庭、クラブ、小さな会社のいずれも、同じカタログを使って作業できます。

## KolleKならではの特徴

**プライベートであること。** あなたのアカウントは自分専用のワークスペースであり、機密情報は保存先で暗号化されます。あなたが共有を選ばない限り、誰もあなたのコレクションを閲覧できません。

**自分で運用できること。** KolleKはオープンソースで、自分でホストする場合は無料です。カタログは自分で管理するインフラ上に存在し、いつでも移動やバックアップができます。

**単なるリストではなく、すべての経緯を記録できること。** 所有する各コピーについて、いくら払ったか、時間の経過による価値、保管場所、誰に貸したか、これまでの手入れの状況、入手元を記録できます。
