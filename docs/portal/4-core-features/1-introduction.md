# Core Features

You know the concepts, now it is time to do the work. This section walks you through the everyday tasks of building and browsing a catalogue: creating collections, adding items and copies, uploading photos, tagging, and keeping an eye on who changed what.

Each page assumes you have read the matching concept page in [Core Concepts](../3-core-concepts/1-introduction.md) and links back to it rather than repeating it.

Read the pages in this order:

1. [Create and manage collections](2-create-and-manage-collections.md). Set up the containers everything else lives in.
2. [Choose how to view a collection](3-choose-how-to-view-a-collection.md). Grid, list, or table, and how in collection search behaves.
3. [Add and edit items](4-add-and-edit-items.md). The central task: putting things into the catalogue.
4. [Track the copies you own](5-track-the-copies-you-own.md). Record each physical copy, its condition, location, and status.
5. [Add photos to an item](6-add-photos-to-an-item.md). Covers and extra shots.
6. [Tag and find items](7-tag-and-find-items.md). Flexible labels that cut across collections.
7. [The activity feed and audit trail](8-activity-feed-and-audit-trail.md). See who changed what, and when.

If you have not catalogued anything yet, the [five minute quick start](../2-getting-started/7-quick-start.md) is the fastest first pass through these tasks.
