# Insights

Depois que você cataloga o suficiente, o KolleK começa a retribuir: quanto vale sua coleção, como ela cresceu e onde o valor está concentrado. Esta seção curta explica as telas que mostram isso.

Leia as páginas nesta ordem:

1. @doc(insights.collectionStatistics). Cada número e gráfico na tela de estatísticas de uma coleção, e de onde vem cada valor.
2. @doc(insights.dashboard). A tela inicial depois que você entra na conta.

As estatísticas são tão boas quanto os dados por trás delas. Se os números parecerem escassos, o ajuste geralmente é registrar @doc(copies.recordPaymentsAndValue, "avaliações e transações") para seus exemplares.
