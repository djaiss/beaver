# Gerencie as tags da conta

As @doc(tags.overview) são as etiquetas compartilhadas da sua conta, e como qualquer vocabulário compartilhado, elas se beneficiam de um pouco de curadoria. Esta página cobre o gerenciamento da lista de tags em si, a partir das configurações da conta, ao contrário de marcar itens um a um, que é coberto em @doc(items.tagAndFind).

Você precisa da função de editor ou proprietário.

## De onde vêm as tags

As tags aparecem na lista de duas formas: alguém cria uma deliberadamente nas configurações da conta, ou alguém digita uma tag nova no formulário de um item e ela é criada na hora. Ambas acabam na mesma lista da conta, compartilhada por todas as coleções.

## Crie, renomeie e exclua

Abra **Tags** nas configurações da conta para ver todas as tags da conta.

- **Crie** uma tag com antecedência quando você quer o vocabulário pronto antes de uma grande sessão de catalogação.
- **Renomeie** uma tag para corrigir um erro de digitação ou unificar termos. Renomear atualiza a etiqueta em todos os lugares onde ela é usada.
- **Exclua** uma tag que você não quer mais.

:::warning
Excluir uma tag a remove de todos os itens da conta de uma vez. Os itens em si não são afetados, mas a etiqueta desaparece de tudo e não há como desfazer a remoção.
:::

## Mantenha a lista útil

Uma lista de tags fica bagunçada silenciosamente: "assinado", "Assinado" e "autografado" significando a mesma coisa. Alguns hábitos mantêm a lista limpa:

- **Combine a grafia com antecedência** se várias pessoas catalogam juntas. Escolha "Assinado" uma vez e mantenha esse padrão.
- **Prefira um punhado de tags significativas** em vez de dezenas de tags usadas uma única vez. Uma tag usada em um único item geralmente é uma anotação, não uma tag.
- **Faça uma limpeza de vez em quando.** Renomeie quase duplicatas em uma só, e exclua tags que não merecem mais o espaço.
- **Não duplique estruturas que já existem em outro lugar.** "Jazz" provavelmente é uma @doc(categories.organizeItems, "categoria"); "Assinado" atravessa tudo e é uma tag perfeita.

Noah mantém exatamente seis tags para o vinil dele: Assinado, Primeira Prensagem, Lacrado, Para Vender, Upgrade da Wishlist e Presente. Cada uma delas responde a uma pergunta que ele realmente faz.

## Para onde ir depois

- Aplique tags enquanto cataloga em @doc(items.tagAndFind).
- Decida entre tags e as outras ferramentas de agrupamento em @doc(organizing.categoriesSetsAndSeries).
