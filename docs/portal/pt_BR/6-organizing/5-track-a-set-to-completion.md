# Acompanhe um set até completá-lo

Um @doc(organizing.categoriesSetsAndSeries, "set") é uma lista finita que você está trabalhando para completar dentro de uma coleção: uma minissérie de doze edições, um time completo de cartas de estreia, uma discografia. Dê a um set uma meta e o KolleK mostra o que você tem contra a meta, então "quão perto eu estou" sempre tem uma resposta.

Você precisa da função de editor ou proprietário para criar ou alterar sets.

## Crie um set

::::steps
:::step title="Abra os sets da coleção"
Abra a coleção e vá até **Sets**.
:::

:::step title="Crie o set"
Escolha **Novo set**, dê um nome a ele e informe a **meta**, o número de itens que o set completo tem. Noah cria "Blue Note série 1500" com uma meta de 100.

::screenshot{label="Formulário de novo set com o campo de meta"}
:::

:::step title="Adicione itens a ele"
Ao adicionar ou editar um item, escolha o set no formulário do item. Cada item que você arquiva no set aumenta o número de itens que você tem.
:::
::::

## Como a conclusão é contada

A página do set soma o que você tem contra a meta. Ter 37 de uma série de 100 discos aparece como 37 de 100. A conclusão também alimenta as @doc(insights.collectionStatistics, "estatísticas da coleção"), onde todos os seus sets se combinam em um número geral de conclusão.

Dois detalhes que vale a pena saber:

- **Só um set com uma meta acima de zero conta para as estatísticas de conclusão.** Um set sem meta ainda agrupa itens, apenas não faz nenhuma afirmação de conclusão.
- **Um set com mais itens do que a meta conta como completo, não como excedente.** Duplicatas e variantes não vão te empurrar além de 100 por cento.

## Sets versus séries

Um set vive dentro de uma coleção e acompanha a conclusão. Uma @doc(series.groupFranchise, "série") atravessa coleções e não acompanha nada; ela só agrupa uma franquia. Se você está se perguntando "quais estão faltando", você quer um set.

## Exclua um set

Excluir um set o envia para a @doc(dataSafety.restoreFromTrash, "lixeira"), e ele pode ser restaurado de lá.

:::warning
Excluir um set remove o agrupamento, e junto com ele o acompanhamento de conclusão. Os itens do set não são excluídos e mantêm todos os seus dados.
:::

## Para onde ir depois

- Agrupe uma franquia entre coleções com @doc(series.groupFranchise, "séries").
- Veja a conclusão sendo somada em @doc(insights.collectionStatistics).
