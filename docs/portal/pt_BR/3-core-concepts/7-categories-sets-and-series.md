# Categorias, sets e séries

O KolleK oferece três formas de agrupar itens, e elas parecem semelhantes até você conhecer a pergunta que cada uma responde. Esta página as desembaraça, para que você use a certa em cada situação.

## As três perguntas

- **Onde este item se arquiva?** Isso é uma **categoria**.
- **Qual lista finita estou tentando completar?** Isso é um **set**.
- **A qual franquia isso pertence, em tudo que eu coleciono?** Isso é uma **série**.

## Categorias: organização dentro de uma coleção

Uma categoria arquiva itens dentro de uma @doc(collections.overview, "coleção"), e categorias podem se aninhar. A coleção "Meus quadrinhos" de Emma tem uma categoria Marvel, com Homem-Aranha e X-Men aninhados dentro dela. Navegar por uma categoria mostra apenas seus itens, com suas próprias contagens e estatísticas.

Uma categoria pertence à sua coleção e não pode alcançar fora dela. Excluir uma categoria nunca exclui os itens dentro dela; eles simplesmente ficam sem categoria.

## Sets: uma lista finita a completar

Um set é uma lista de verificação com um fim. Emma quer as dez primeiras edições de uma série, então ela cria um set com uma **meta** de dez. Conforme ela adiciona itens ao set, o KolleK mostra o quanto ela possui em relação à meta, e a conclusão entra nas suas @doc(insights.collectionStatistics, "estatísticas da coleção").

Assim como as categorias, um set pertence a uma coleção. Somente um set com meta acima de zero conta para as estatísticas de conclusão; um set sem meta é apenas um agrupamento nomeado.

## Séries: uma franquia entre coleções

Uma série reúne tudo de uma franquia, onde quer que esteja. Harry Potter se espalha pelos livros de Emma, seus filmes e seu LEGO, que são três coleções diferentes. Uma série alcança todas elas, e sua página mostra os itens vinculados agrupados pela coleção em que cada um vive, junto com quantas coleções ela alcança.

Uma série não acompanha conclusão. Ela responde "mostre-me tudo de Harry Potter", não "já terminei?".

## Lado a lado

| | Categoria | Set | Série |
|---|---|---|---|
| Escopo | Uma coleção | Uma coleção | A conta inteira |
| Se aninha | Sim | Não | Não |
| Acompanha conclusão | Não | Sim, com uma meta | Não |
| Pergunta que responde | Onde isso se arquiva | De qual lista isso faz parte | A qual franquia isso pertence |

Um item pode usar as três ao mesmo tempo: arquivado em Marvel, parte do set "Dez primeiras edições" e vinculado à série Homem-Aranha.

Se você quiser algo ainda mais leve que essas, rótulos que cruzam tudo sem nenhuma estrutura, é para isso que servem as @doc(tags.overview, "tags").
