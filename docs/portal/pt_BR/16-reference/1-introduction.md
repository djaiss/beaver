# Referência

As páginas desta seção são material de consulta. Elas foram feitas para serem escaneadas, não lidas de cima a baixo. Quando outra página menciona um termo, uma opção ou um e-mail sobre o qual você quer o quadro completo, a resposta está aqui.

1. @doc(reference.glossary). Todo termo do produto, definido em uma ou duas linhas, com um link para a página que o explica completamente.
2. @doc(reference.fieldAndStatus). Todo conjunto de opções que você encontra em um formulário, de status de exemplar a tipos de evento de procedência, com um significado de uma linha para cada.
3. @doc(reference.emailsSent). Todo e-mail que o sistema pode enviar, o que o dispara e quem o recebe.

Se você é novo no KolleK, não comece por aqui. Comece por @doc(gettingStarted.index), e volte quando precisar consultar alguma coisa.
