# Desenvolvedores e a API

O KolleK oferece uma API JSON completa que espelha o aplicativo web: tudo o que você pode fazer no aplicativo, você pode fazer com um token e um cliente HTTP. Esta seção te deixa autenticado e orientado. A referência completa de endpoints é gerada a partir do código e está sempre atualizada, então estas páginas deliberadamente não a repetem.

Leia as páginas nesta ordem:

1. @doc(api.overview). O que a API cobre, como ela é estruturada, e onde fica a referência gerada.
2. @doc(api.authenticate). Obtenha um token e faça sua primeira requisição.
3. @doc(api.rateLimitsAndConventions). Paginação, dinheiro, limitação de taxa e comportamento de erros.
4. @doc(webhooks.overview). Registre endpoints hoje, e entenda exatamente o que dispara (e o que ainda não dispara).

Para detalhes de cada endpoint (parâmetros, formatos de resposta, exemplos), use a referência gerada em `/docs/api` na sua instância.
