# Escolha como visualizar uma coleção

Toda coleção pode ser navegada de três formas, e o KolleK lembra qual você prefere para cada uma. Esta página cobre os layouts, o que o cabeçalho da coleção mostra, e exatamente como funciona a pesquisa dentro de uma coleção.

## Os três layouts

Abra uma coleção e use o seletor de layout para alternar entre:

- **Grade.** Fotos de capa em grade. Ótimo para coleções visuais, onde você reconhece as coisas pela capa. Este é o padrão.
- **Lista.** Linhas compactas, mais itens na tela. Bom para uma varredura rápida.
- **Tabela.** Uma coluna por detalhe. Ótimo quando você quer comparar detalhes entre itens.

Sua escolha é lembrada **por usuário, por coleção**. Emma pode navegar pelos seus gibis em grade enquanto Sam, na mesma conta, vê a mesma coleção como tabela. Um não afeta o outro, e nenhum dos dois afeta outras coleções.

## O que o cabeçalho mostra

No topo da coleção você verá sua **quantidade de itens** e seu **valor total**. O valor total é a soma do valor atual de cada exemplar dentro dela, onde o valor atual de cada exemplar vem da sua @doc(copies.recordPaymentsAndValue, "avaliação") mais recente. Exemplares que nunca foram avaliados não contribuem em nada, então um total baixo geralmente significa avaliações faltando, não itens sem valor.

## Pesquisando dentro de uma coleção

A caixa de pesquisa acima dos itens é um filtro rápido, e vale a pena entender o que ela faz e o que não faz.

Ela reduz os itens **carregados na página no momento** enquanto você digita. Uma página de coleção carrega até 1000 itens de uma vez, então para a maioria das coleções o filtro efetivamente cobre tudo. Se sua coleção for maior que isso, o filtro só enxerga a página carregada.

:::note
Este é um filtro do que está visível, não uma pesquisa global. Ele olha para a coleção que você tem aberta, não para toda a sua conta.
:::

Uma pesquisa global em todas as coleções está planejada, mas ainda não está disponível. Veja a @doc(troubleshooting.featureStatus, "página de status de recursos") para saber como isso está.
