# O que é o KolleK

O KolleK é um lar para tudo o que você coleciona. Quadrinhos, discos de vinil, moedas, relógios, vinhos, livros, cartas colecionáveis, video games, ou aquela coisa estranhamente específica que só você ama. Se você pode possuir, você pode catalogar aqui.

## O problema que ele resolve

A maioria dos colecionadores acaba espalhada entre uma planilha, um aplicativo de notas, algumas pastas de fotos e a própria memória. Funciona até a coleção crescer. Depois disso, você não consegue mais lembrar quanto pagou, em qual prateleira uma peça está, se uma delas está emprestada a um amigo, ou quanto vale o conjunto todo.

O KolleK substitui tudo isso por uma única ferramenta focada, feita para coleções.

## Como ele é organizado

Você só precisa entender uma ideia para começar. Tudo no KolleK se encaixa em quatro níveis:

- Uma **@doc(accounts.usersAndRoles, "conta")** é seu espaço de trabalho. Tudo o que você cria vive dentro dela.
- Uma **@doc(collections.overview, "coleção")** é um grupo nomeado, como "Meus Quadrinhos" ou "Adega de Vinhos".
- Um **@doc(items.itemsVsCopies, "item")** é um tipo de coisa, como "Amazing Spider-Man #1".
- Um **@doc(items.itemsVsCopies, "exemplar")** é uma coisa física que você realmente possui.

A distinção entre um item e um exemplar é o que torna o KolleK diferente de uma lista comum. Se você possui três exemplares do mesmo quadrinho, isso é um item com três exemplares, e cada exemplar tem sua própria condição, local de armazenamento, valor e histórico. Você vai conhecer essa ideia de verdade assim que começar a catalogar, então não se preocupe com isso ainda.

:::note
A diferença entre um item e um exemplar é a única ideia que vale a pena levar adiante, e você vai usá-la o tempo todo.
:::

## Para quem é

O KolleK funciona tanto para um colecionador solo mantendo um catálogo privado quanto para um grupo compartilhando um. Você pode convidar outras pessoas para sua conta e decidir se cada uma pode visualizar, editar ou gerenciar. Uma família, um clube ou uma pequena empresa podem trabalhar todos a partir do mesmo catálogo.

## O que o torna diferente

**É privado.** Sua conta é seu próprio espaço de trabalho, e detalhes sensíveis são criptografados onde são armazenados. Ninguém navega pela sua coleção a menos que você escolha compartilhá-la.

**É seu para administrar.** O KolleK é de código aberto e gratuito para autohospedar. Seu catálogo vive na infraestrutura que você controla, e você pode movê-lo ou fazer backup dele quando quiser.

**Ele acompanha a história completa, não apenas uma lista.** Para cada exemplar que você possui, você pode registrar quanto pagou, quanto ele vale ao longo do tempo, onde está guardado, para quem você emprestou, os cuidados que ele teve, e de onde veio.
