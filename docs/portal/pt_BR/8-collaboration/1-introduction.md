# Colaboração

Uma conta do KolleK é um espaço de trabalho compartilhado. Você pode mantê-la só para você, ou trazer sua família, seu clube ou seus sócios, cada um com o nível certo de acesso. Esta seção cobre como trazer pessoas para dentro, decidir o que elas podem fazer, e compartilhar uma coleção com o mundo exterior.

Leia as páginas nesta ordem:

1. @doc(collaboration.invitePeople). Envie um convite e saiba o que a pessoa convidada vai vivenciar.
2. @doc(collaboration.manageMembersAndRoles). Altere o que membros existentes podem fazer, ou remova-os.
3. @doc(collaboration.rolesInPractice). Uma referência fácil de consultar sobre quem pode fazer o quê.
4. @doc(collections.share). Torne uma coleção visível para sua conta, ou para qualquer pessoa com um link.

Se você ainda não leu a página de conceito sobre @doc(accounts.usersAndRoles, "contas, usuários e funções"), comece por ela. Ela explica o modelo sobre o qual estas páginas se apoiam.
