# Tutorials

Die Konzept- und Anleitungsseiten anderswo in dieser Dokumentation vermitteln jeweils eine Sache auf einmal. Tutorials fügen diese Teile zu vollständigen, realistischen Reisen zusammen. Jedes folgt einem glaubwürdigen Sammler von Anfang bis Ende, erklärt, warum jeder Schritt wichtig ist, und zeigt dir, was du sehen solltest, wenn es funktioniert hat.

Wähle das Tutorial, das zu deiner Situation passt:

1. @doc(tutorials.catalogueFirstCollection, "Deine erste Sammlung von Anfang bis Ende katalogisieren"). Emma verwandelt ein leeres Konto in eine echte Comics-Sammlung mit einem Objekt, einem Foto, einem Exemplar, einem Kauf und einer Bewertung.
2. @doc(tutorials.setupForHobby, "Dein Konto für ein bestimmtes Hobby einrichten"). Noah passt Typen, Felder, Standorte und Tags für seine Schallplatten an, bevor er die Masse seiner Daten erfasst.
3. @doc(tutorials.trackValuableItem, "Das ganze Leben eines wertvollen Objekts nachverfolgen"). Priya begleitet eine antike Uhr durch Kauf, Begutachtung, Versicherung, Papierkram, eine Leihgabe für eine Ausstellung, Wartung und Herkunftsnachweis.
4. @doc(tutorials.inviteHousehold, "Deinen Haushalt oder Verein einladen und Berechtigungen festlegen"). Emma holt zwei Personen mit unterschiedlichen Rollen in ihr Konto und teilt eine Sammlung sicher.
5. @doc(tutorials.selfHostWithDocker, "KolleK mit Docker selbst hosten"). Alex installiert eine funktionierende Instanz von Grund auf und richtet den ersten Instanzadministrator ein.

Wenn du KolleK noch nie benutzt hast, beginne mit dem ersten Tutorial oder mit dem kürzeren @doc(gettingStarted.quickStart, "Fünf-Minuten-Schnellstart"), wenn du nur einen Moment Zeit hast.
