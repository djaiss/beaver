# Referenz

Die Seiten in diesem Abschnitt sind Nachschlagematerial. Sie sind zum Überfliegen gedacht, nicht zum Durchlesen von vorne bis hinten. Wenn eine andere Seite einen Begriff, eine Option oder eine E-Mail erwähnt, zu der du das vollständige Bild willst, findest du die Antwort hier.

1. @doc(reference.glossary). Jeder Produktbegriff, in ein bis zwei Zeilen erklärt, mit einem Link zur Seite, die ihn vollständig behandelt.
2. @doc(reference.fieldAndStatus). Jede Optionsliste, der du in einem Formular begegnest, von Exemplarstatus bis Provenienz-Ereignistypen, mit einer kurzen Bedeutung für jede.
3. @doc(reference.emailsSent). Jede E-Mail, die das System versenden kann, was sie auslöst, und wer sie erhält.

Wenn du neu bei KolleK bist, fang nicht hier an. Beginne mit @doc(gettingStarted.index) und komm hierher zurück, wenn du etwas nachschlagen musst.
