# Ein Set bis zur Vollständigkeit verfolgen

Ein @doc(organizing.categoriesSetsAndSeries, "Set") ist eine endliche Liste, die du innerhalb einer Sammlung vervollständigen willst: eine Miniserie mit zwölf Heften, ein komplettes Team an Rookie-Karten, eine Diskografie. Gib einem Set eine Zielanzahl, und KolleK zeigt dir Besitz gegen Ziel, sodass die Frage "wie nah bin ich dran" immer eine Antwort hat.

Du brauchst die Rolle Bearbeiter oder Eigentümer, um Sets zu erstellen oder zu ändern.

## Ein Set erstellen

::::steps
:::step title="Die Sets der Sammlung öffnen"
Öffne die Sammlung und gehe zu ihren **Sets**.
:::

:::step title="Das Set erstellen"
Wähle **Neues Set**, benenne es und gib die **Zielanzahl** ein, also die Anzahl an Objekten, die das vollständige Set umfasst. Noah erstellt "Blue Note 1500 Series" mit einem Ziel von 100.

::screenshot{label="Formular für ein neues Set mit dem Feld für die Zielanzahl"}
:::

:::step title="Objekte hinzufügen"
Wähle beim Hinzufügen oder Bearbeiten eines Objekts das Set im Objektformular. Jedes Objekt, das du in das Set einordnest, erhöht die besessene Anzahl.
:::
::::

## Wie die Vollständigkeit gezählt wird

Die Set-Seite stellt gegenüber, was du besitzt, verglichen mit dem Ziel. Besitzt du 37 von einer 100er-Reihe, liest sich das als 37 von 100. Die Vollständigkeit fließt auch in die @doc(insights.collectionStatistics, "Sammlungsstatistik") ein, wo sich alle deine Sets zu einer Gesamtvollständigkeit summieren.

Zwei Details, die du kennen solltest:

- **Nur ein Set mit einer Zielanzahl über null zählt zur Vollständigkeitsstatistik.** Ein Set ohne Ziel gruppiert Objekte trotzdem, macht aber keine Aussage zur Vollständigkeit.
- **Ein Set, das mehr Objekte enthält als sein Ziel, gilt als vollständig, nicht als übervollständig.** Duplikate und Varianten bringen dich nicht über 100 Prozent hinaus.

## Sets im Vergleich zu Reihen

Ein Set lebt innerhalb einer Sammlung und verfolgt die Vollständigkeit. Eine @doc(series.groupFranchise, "Reihe") umfasst mehrere Sammlungen und verfolgt nichts, sie gruppiert nur ein Franchise. Fragst du dich "welche fehlen mir noch", willst du ein Set.

## Ein Set löschen

Löschst du ein Set, wandert es in den @doc(dataSafety.restoreFromTrash, "Papierkorb") und kann von dort wiederhergestellt werden.

:::warning
Löschst du ein Set, wird die Gruppierung entfernt, und die Vollständigkeitsverfolgung mit ihr. Die Objekte im Set werden nicht gelöscht und behalten all ihre Daten.
:::

## Wie es weitergeht

- Gruppiere ein Franchise über Sammlungen hinweg mit @doc(series.groupFranchise, "Reihen").
- Sieh, wie sich die Vollständigkeit summiert, in @doc(insights.collectionStatistics).
