# Sichtbarkeit und Teilen

Jede Sammlung trägt eine Sichtbarkeitseinstellung, die festhält, für wen sie gedacht ist. Diese Seite erklärt die drei Stufen und, genauso wichtig, wie viel davon heute tatsächlich durchgesetzt wird, damit dich nichts in beide Richtungen überrascht.

## Die drei Stufen

Die Sichtbarkeit wird für jede @doc(collections.overview, "Sammlung") einzeln festgelegt, sodass ein Konto alle drei mischen kann:

- **Privat.** Nur für dich gedacht. Die richtige Wahl für alles Persönliche oder für einen Katalog, der noch aufgebaut wird.
- **Geteilt.** Für jedes Mitglied deines @doc(accounts.usersAndRoles, "Kontos") gedacht: Eigentümer, Bearbeiter und Betrachter gleichermaßen.
- **Öffentlich.** Für jeden gedacht, dem du den Link der Sammlung gibst, nur lesend, ohne Anmeldung.

Priya markiert "Geerbte Stücke" als privat, "Uhren" als geteilt mit ihrer Familie und den Vereinskatalog, mit dem sie prahlen möchte, als öffentlich.

## Wie viel heute durchgesetzt wird

Sichtbarkeit ist aktuell eine erfasste Präferenz, noch keine funktionierende Sperre. Zwei Dinge folgen daraus, jeweils in eine Richtung:

- **Innerhalb des Kontos kann jedes Mitglied jede Sammlung durchsehen**, unabhängig von ihrer Sichtbarkeit. Eine Sammlung als privat zu markieren, verbirgt sie noch nicht vor den anderen Mitgliedern deines Kontos.
- **Von außen ist nichts erreichbar.** Es gibt noch keine öffentliche Sammlungsseite, sodass eine als öffentlich markierte Sammlung keinen Link hat, den man herausgeben könnte, und niemand außerhalb deines Kontos sie sehen kann. In diesem Sinn sind deine Daten sogar privater, als es die Einstellung vermuten lässt.

Die Sichtbarkeit heute festzulegen, hält deine Absicht fest, sodass sich dein Katalog richtig verhält, sobald Teilen verfügbar wird. Die gepflegte Übersicht zu diesem Status ist @doc(troubleshooting.featureStatus).

:::note
Teilst du ein Konto mit anderen Personen, geh davon aus, dass sie vorerst jede Sammlung darin sehen können, auch private. Behalte alles wirklich Persönliche in einem eigenen Konto.
:::

## Was öffentlich bedeuten wird

Wenn öffentliche Links verfügbar werden, kann eine öffentliche Sammlung von jedem eingesehen werden, der ihren Link hat, von überall, ohne Anmeldung. Das verdient eine bewusste Entscheidung, weshalb die Einstellung für jede Sammlung existiert, statt als ein großer Schalter.

## Sichtbarkeit und Rollen wirken zusammen

Sichtbarkeit dreht sich ums Sehen. @doc(accounts.usersAndRoles, "Rollen") drehen sich ums Ändern. Ein Betrachter in deinem Konto kann durchsehen, aber nichts bearbeiten, unabhängig von der Sichtbarkeit einer Sammlung. Ein öffentlicher Besucher wird gar kein Mitglied sein, sodass keine Rolle gilt und nichts bearbeitbar ist.

## Wie geht es weiter

- Lege die Sichtbarkeit einer Sammlung Schritt für Schritt fest in @doc(collections.share).
- Sieh nach, wer in deinem Konto ist, in @doc(collaboration.manageMembersAndRoles).
- Fragst du dich, wie deine Daten gespeichert werden? Siehe @doc(dataSafety.howProtected).
