# Core Features

You know the concepts, now it is time to do the work. This section walks you through the everyday tasks of building and browsing a catalogue: creating collections, adding items and copies, uploading photos, tagging, and keeping an eye on who changed what.

Each page assumes you have read the matching concept page in [Core Concepts](../core-concepts/README.md) and links back to it rather than repeating it.

Read the pages in this order:

1. [Create and manage collections](create-and-manage-collections.md). Set up the containers everything else lives in.
2. [Choose how to view a collection](choose-how-to-view-a-collection.md). Grid, list, or table, and how in collection search behaves.
3. [Add and edit items](add-and-edit-items.md). The central task: putting things into the catalogue.
4. [Track the copies you own](track-the-copies-you-own.md). Record each physical copy, its condition, location, and status.
5. [Add photos to an item](add-photos-to-an-item.md). Covers and extra shots.
6. [Tag and find items](tag-and-find-items.md). Flexible labels that cut across collections.
7. [The activity feed and audit trail](activity-feed-and-audit-trail.md). See who changed what, and when.

If you have not catalogued anything yet, the [five minute quick start](../getting-started/quick-start.md) is the fastest first pass through these tasks.
