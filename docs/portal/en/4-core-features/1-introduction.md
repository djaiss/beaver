# Core Features

You know the concepts, now it is time to do the work. This section walks you through the everyday tasks of building and browsing a catalogue: creating collections, adding items and copies, uploading photos, tagging, and keeping an eye on who changed what.

Each page assumes you have read the matching concept page in @doc(coreConcepts.index) and links back to it rather than repeating it.

Read the pages in this order:

1. @doc(collections.manage). Set up the containers everything else lives in.
2. @doc(collections.chooseView). Grid, list, or table, and how in collection search behaves.
3. @doc(items.addAndEdit). The central task: putting things into the catalogue.
4. @doc(copies.track). Record each physical copy, its condition, location, and status.
5. @doc(items.addPhotos). Covers and extra shots.
6. @doc(items.tagAndFind). Flexible labels that cut across collections.
7. @doc(activity.feedAndAuditTrail). See who changed what, and when.

If you have not catalogued anything yet, the @doc(gettingStarted.quickStart, "five minute quick start") is the fastest first pass through these tasks.
