# 参考手册

本节的页面是用来查阅的资料，不需要从头读到尾。当其他页面提到某个术语、某个选项，或者某封你想全面了解的邮件时，答案就在这里。

1. @doc(reference.glossary)。每个产品术语都有一到两行的定义，并附有指向完整说明页面的链接。
2. @doc(reference.fieldAndStatus)。你在表单中会遇到的每一组选项，从副本状态到来源事件类型，每项都配有一行简明含义。
3. @doc(reference.emailsSent)。系统可能发送的每一种邮件，触发条件及接收对象。

如果你刚接触 KolleK，不要从这里开始。请先阅读 @doc(gettingStarted.index)，等你需要查阅某项内容时再回到这里。
