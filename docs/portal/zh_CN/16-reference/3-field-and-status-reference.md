# 字段与状态参考

汇总你在 KolleK 表单中会遇到的每一组选项，方便一次查阅。每组都链接到使用它的指南。关于术语本身的定义，请参见@doc(reference.glossary, "术语表")。

## 副本状态

在你记录的每个副本上设置。用于 @doc(copies.track)。

| 状态 | 含义 |
| --- | --- |
| 持有中 | 你持有这个副本，新建副本的默认状态。 |
| 已订购 | 已购买或已预订，正在寄送给你的途中。 |
| 已借出 | 目前在别人手中，保管权转移了，所有权没有变。 |
| 已出售 | 你已卖出，不再拥有它。 |
| 已赠出 | 你把它送给了别人。 |
| 已丢失 | 你找不到它，也不指望再找到。 |
| 已被盗 | 从你这里被偷走。 |
| 已处置 | 已丢弃或回收，可选填处置日期。 |
| 其他 | 以上都不适用的情况。 |

:::note
持有中、已订购、已借出都算作仍然持有。借出的副本仍然是你的，只是暂时在别处。
:::

## 交易类型

在每笔交易上设置。用于 @doc(copies.recordPaymentsAndValue)。标记为"取得"的类型代表副本进入了你手中，最早的取得类交易会作为副本的取得日期。

| 类型 | 含义 |
| --- | --- |
| 购买 | 你购买了该副本。属于取得。 |
| 出售 | 你卖出了该副本。 |
| 交换 | 你用其他东西换来了它。属于取得。 |
| 收到赠礼 | 别人把它送给了你。属于取得。 |
| 赠出礼物 | 你把它送给了别人。 |
| 继承 | 它通过继承归你所有。属于取得。 |
| 退款 | 之前某笔交易的退款。 |
| 费用 | 与副本相关的开销，例如拍卖费。 |
| 税费 | 就该副本缴纳的税款。 |
| 运费 | 单独记录的配送费用。 |
| 其他 | 以上都不适用的金钱事件。 |

## 估值类型与置信度

在每次估值上设置。用于 @doc(copies.recordPaymentsAndValue)。

| 估值类型 | 含义 |
| --- | --- |
| 自行估计 | 你自己对价值的判断。 |
| 专业鉴定 | 由专业人士给出的正式鉴定。 |
| 市场估计 | 根据当前市场或成交数据推算得出。 |
| 保险价值 | 用于保险用途的价值。 |
| 拍卖估价 | 拍卖行给出的估价。 |
| 自动估算 | 由定价服务或工具生成。 |
| 其他 | 其他估值依据。 |

| 置信度 | 含义 |
| --- | --- |
| 低 | 粗略猜测。 |
| 中 | 有一定依据，较为合理。 |
| 高 | 有充分依据，例如近期的专业鉴定。 |
| 未知 | 未记录置信度。 |

## 保险状态

在每条保险记录上设置。用于 @doc(copies.insure)。保险记录上的保险类型为自由文本，没有固定选项列表。

| 状态 | 含义 |
| --- | --- |
| 生效中 | 保单目前正为该副本提供保障。 |
| 已过期 | 保障期已结束。 |
| 已取消 | 保单在到期前被取消。 |
| 待生效 | 保障已安排但尚未生效。 |

## 借还方向与状态

在每条借还记录上设置。用于 @doc(loans.lendAndBorrow)。

| 方向 | 含义 |
| --- | --- |
| 借出 | 你的副本离开了你手中，例如借给朋友或用于展览。 |
| 借入 | 别人的物品目前在你手中。 |

| 状态 | 含义 |
| --- | --- |
| 已计划 | 已谈妥但尚未交接。 |
| 进行中 | 副本目前处于外借（或外借入）状态。 |
| 已逾期 | 超过约定归还日期仍未归还。KolleK 会每天自动标记此状态。 |
| 已归还 | 借还已结束，副本已归还。 |
| 已取消 | 该次借还并未实际发生。 |
| 已丢失 | 副本未能归还。 |

## 保养类型

在每条保养记录上设置。用于 @doc(copies.recordMaintenance)。

| 类型 | 含义 |
| --- | --- |
| 清洁 | 常规清洁。 |
| 维修 | 修复损坏。 |
| 保养 | 定期维护，例如手表保养。 |
| 保存处理 | 用于稳定和保护状态的处理。 |
| 修复还原 | 使副本恢复到较早状态的处理。 |
| 部件更换 | 更换零件或组件。 |
| 检查 | 不进行干预的检查。 |

## 来源事件类型与日期精度

在每条来源事件上设置。用于 @doc(copies.traceProvenance)。

| 事件类型 | 含义 |
| --- | --- |
| 取得 | 副本进入某个收藏。 |
| 出售 | 副本被卖出。 |
| 赠予 | 副本作为礼物易手。 |
| 继承 | 副本通过遗产继承转手。 |
| 所有权转移 | 所有权以其他方式发生变化。 |
| 保管权转移 | 副本移动但所有权未变。 |
| 借出 | 副本被外借。 |
| 归还 | 副本从借出状态归还。 |
| 展览 | 副本曾公开展出。 |
| 鉴定验真 | 副本被验证为真品。 |
| 评估鉴定 | 副本经过正式估值。 |
| 重大修复 | 值得记入历史的重要修复工作。 |
| 起源 | 副本的制作时间和地点。 |
| 发现 | 副本被找到或重新发现。 |
| 其他 | 故事中的其他章节。 |

来源日期往往并不确定，因此每个事件都带有一个精度标记：

| 精度 | 含义 |
| --- | --- |
| 精确日期 | 完整日期已知。 |
| 精确到月 | 精确到某个月份。 |
| 精确到年 | 精确到某一年。 |
| 大致估计 | 一个最佳估计，可理解为"约"。 |
| 未知 | 未记录日期。 |

## 文档类型

在每份文档上设置。用于 @doc(copies.attachDocuments)。

| 类型 | 含义 |
| --- | --- |
| 收据 | 购买凭证。 |
| 发票 | 副本或其相关工作的账单。 |
| 证书 | 随副本附带的证书。 |
| 鉴定报告 | 书面估值报告。 |
| 保险单据 | 保单相关文件。 |
| 照片 | 作为记录而非展示图保存的照片。 |
| 品相报告 | 品相评估的书面报告。 |
| 修复报告 | 修复工作的记录。 |
| 目录条目 | 目录条目或著录信息。 |
| 往来信件 | 与该副本相关的信件或邮件。 |
| 所有权凭证 | 证明所有权的文件。 |
| 真伪凭证 | 证明副本为真品的文件。 |
| 其他 | 其他值得保存的文件。 |

## 自定义字段类型

在收藏类型上定义自定义字段时选择。用于 @doc(collectionTypes.setup)。

| 字段类型 | 含义 |
| --- | --- |
| 文本 | 自由文本，例如作者或出版社。 |
| 数字 | 数值，例如期号。 |
| 日期 | 日历日期，例如发行日期。 |
| 是 / 否 | 一个复选框，例如"已签名"。 |
| 单选 | 从你定义的选项列表中选择一项。 |
| 评分 | 星级评分，最多五星。 |

## 收藏可见性

在每个收藏上设置。用于 @doc(collections.share)。此设置目前只是记录状态，等分享功能上线后才会强制生效；参见 @doc(troubleshooting.featureStatus)。

| 可见性 | 含义 |
| --- | --- |
| 私密 | 仅供你自己查看。 |
| 共享 | 面向账户内所有人。 |
| 公开 | 任何拥有链接的人都可查看，无需登录，只读。 |
