# 教程

文档中其他章节的概念页面和操作指南每次只讲解一个知识点，而教程会把这些内容串联成完整、贴近真实场景的操作流程。每篇教程都会从头到尾跟随一位真实可信的收藏者，解释每一步为什么重要，并告诉你操作成功后应该看到什么效果。

请根据你目前的情况选择合适的教程：

1. @doc(tutorials.catalogueFirstCollection, "端到端编目你的第一个收藏")。Emma 从一个空账户出发，建立起一个真实的漫画收藏，其中包含一件藏品、一张照片、一个副本、一笔购买记录和一次估值。
2. @doc(tutorials.setupForHobby, "为特定爱好配置你的账户")。Noah 在批量录入数据之前，为自己的黑胶唱片收藏定制了类型、字段、位置和标签。
3. @doc(tutorials.trackValuableItem, "追踪一件贵重藏品的完整生命周期")。Priya 记录了一块古董手表从购买、鉴定、投保、相关文件、展览外借、保养到来源信息的完整历程。
4. @doc(tutorials.inviteHousehold, "邀请你的家人或俱乐部成员并设置权限")。Emma 以不同角色邀请两位成员加入她的账户，并安全地共享了一个收藏。
5. @doc(tutorials.selfHostWithDocker, "使用 Docker 自托管 KolleK")。Alex 从零开始搭建了一个可运行的实例，并授予了第一个实例管理员权限。

如果你从未使用过 KolleK，建议从第一篇教程开始；如果时间有限，也可以先阅读更简短的 @doc(gettingStarted.quickStart, "五分钟快速上手")。
