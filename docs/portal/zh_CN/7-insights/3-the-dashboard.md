# 仪表盘

仪表盘是你登录后看到的第一个界面，它是你账户状态的一份简短报告：都有谁在，以及最近发生了什么。

## 它展示什么

- **问候语。** 根据时间显示早上好、下午好，或晚上好，并带上你的名字。
- **成员和邀请。** 账户中有多少人，还有多少@doc(collaboration.invitePeople, "邀请")尚未被接受。
- **最近活动。** 整个账户中所有成员的最新操作，包括谁做了什么、多久之前做的。这是@doc(activity.feedAndAuditTrail, "活动动态与审计记录")的前台展示窗口。

从这里出发，你的各个收藏在导航栏中始终一键可达。

## 搜索框

仪表盘上有一个搜索框，用来在你整个账户范围内查找藏品。全局搜索功能目前还不可用，这个搜索框只是一个占位符。在单个收藏内部搜索已经可以使用了，具体说明见@doc(collections.chooseView)。全局搜索的当前进展，请参见@doc(troubleshooting.featureStatus, "功能状态页面")。

## 全新账户会跳过它

一个还没有任何收藏的全新账户，登录后会被带到@doc(gettingStarted.checklist, "新手上手清单")，而不是仪表盘。一旦第一个收藏被创建（或者所有者隐藏了这份清单），仪表盘就会接管，成为登录后的落地界面。

## 接下来可以做什么

- 深入了解某个收藏的数字，参见@doc(insights.collectionStatistics)。
- 了解活动动态，参见@doc(activity.feedAndAuditTrail)。
- 邀请其他人加入，参见@doc(collaboration.invitePeople)。
