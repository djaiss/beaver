# 洞察

当你编目的藏品足够多之后，KolleK 就开始给你回报：你的收藏值多少钱、它是如何增长的、价值又集中在哪里。这一小节将介绍展示这些信息的界面。

请按以下顺序阅读：

1. @doc(insights.collectionStatistics)：收藏统计界面上每一个数字和图表的含义，以及它们各自的数据来源。
2. @doc(insights.dashboard)：登录后看到的首页。

统计结果的好坏取决于背后的数据。如果数字看起来单薄，通常的解决办法是为你的副本记录@doc(copies.recordPaymentsAndValue, "估值和交易")。
