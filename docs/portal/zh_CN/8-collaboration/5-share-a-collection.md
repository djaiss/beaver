# 分享收藏

每个收藏都带有一项可见性设置，用来记录它面向谁。本页将介绍如何设置它，并如实说明这项设置目前的实际效果，因为分享功能在 KolleK 中仍在不断完善之中。

三个级别在@doc(sharing.overview, "可见性与分享")中有完整说明：

- **私密。** 仅供你自己查看。
- **共享。** 面向账户内所有人。
- **公开。** 任何拥有链接的人都可查看，无需登录，只读。

:::note
目前这项设置只是记录了你的意图，尚未强制生效。账户内的每个成员仍然可以浏览所有收藏，也还没有可以分发的公开链接，因此目前没有任何内容可以从账户之外访问。当前的进展状态可在 @doc(troubleshooting.featureStatus) 中查看。
:::

## 修改收藏的可见性

::::steps
:::step title="打开收藏设置"
打开某个@doc(collections.overview, "收藏")并选择编辑它。可见性设置就在收藏本身的编辑页面上，与名称、表情符号在一起。
:::

:::step title="选择可见性"
选择**私密**、**共享**或**公开**。

::screenshot{label="收藏表单，含三种可见性选项"}
:::

:::step title="保存"
保存收藏，设置会立即生效。
:::
::::

Emma 把"心愿单调研"这个收藏标记为私密，把"我的漫画"标记为对 Sam 和 Leo 共享，把"复古海报"标记为公开，为将来能够分享链接做好准备。

## 为什么要现在就设置

现在就设置好可见性，意味着一旦分享功能上线，你的目录会立即以正确的方式运作。你标记为私密的收藏会保持私密，你标记为公开的收藏会自动获得链接，你不需要在功能上线后再逐个重新检查每个收藏。

:::warning
公开链接功能上线后，任何拥有链接的人都可以查看被标记为公开的收藏，无需登录，包括其中的每一件藏品、照片和细节。如果你不希望这些信息被外部看到，尤其是当它会暴露你拥有什么以及居住在哪里时，请不要将该收藏标记为公开。
:::

## 重新设为私密

随时可以编辑收藏，将其可见性改回**私密**（或**共享**）。设置会从那一刻起对该收藏生效。

:::note
分享设置控制的是谁能看到某个收藏。成员对收藏能做什么，是由其角色决定的，而不是由可见性决定的。查看者无法编辑任何内容，编辑者则可以。参见@doc(collaboration.rolesInPractice, "三种角色的实际运作")。
:::
