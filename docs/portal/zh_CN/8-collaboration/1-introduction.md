# 协作

一个 KolleK 账户是一个共享的工作空间。你可以把它留给自己一个人使用，也可以邀请你的家人、俱乐部成员，或者生意伙伴加入，并为每个人设置合适的访问权限。本节将介绍如何邀请他人、决定他们能做什么，以及如何将收藏分享给外部人员。

请按以下顺序阅读各页面：

1. @doc(collaboration.invitePeople)。发送邀请，并了解被邀请人会看到什么。
2. @doc(collaboration.manageMembersAndRoles)。修改现有成员的权限，或将其移除。
3. @doc(collaboration.rolesInPractice)。一份便于查阅的权限对照表。
4. @doc(collections.share)。让某个收藏对账户内所有人可见，或对任何拥有链接的人可见。

如果你还没有阅读关于@doc(accounts.usersAndRoles, "账户、用户与角色")的概念说明页面，建议从那里开始，它解释了本节内容所基于的整体模型。
