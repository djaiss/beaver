# 设置收藏类型和自定义字段

@doc(collectionTypes.overview, "收藏类型") 决定了一件藏品可以记录哪些信息。漫画需要期号和出版社，黑胶唱片需要艺术家和压制版本。本页介绍如何创建一个类型、为它添加自定义字段，并用字段分组让较长的表单保持清晰易读。

管理类型需要编辑者或所有者角色。类型是账户级别共享的，一次精心设置好的类型可以被任意数量的收藏重复使用。

## 从现成的类型开始

一个新账户已经内置了十几种现成类型（漫画、集换式卡牌、黑胶唱片、CD、DVD、硬币、邮票、图书、手办 / 玩具、电子游戏、手表和红酒），每种都已经配好了合理的字段分组。与其从零开始，不如先打开最接近你爱好的那个类型，在此基础上调整。重命名或新增一个字段，总比从空白开始要快。

## 创建一个类型

Noah 收藏黑胶唱片，他想为演唱会海报建一个类型，这是现成类型里没有覆盖到的。

::::steps
:::step title="打开收藏类型"
进入账户设置，打开**收藏类型**。

::screenshot{label="账户设置中的收藏类型列表"}
:::

:::step title="创建类型"
选择**新建类型**，输入名称（Noah 输入了“演唱会海报”），再选一个颜色。颜色能帮你在列表中一眼分辨不同类型。
:::

:::step title="添加第一批字段"
打开新类型，为你想记录的每项信息添加一个自定义字段。每个字段都要选择名称和字段类型。

::screenshot{label="带字段列表的类型编辑器"}
:::
::::

类型编辑器会随手保存，没有单独的保存按钮需要记住，每次修改都会立即存储。

## 选择合适的字段类型

每个自定义字段都属于以下六种字段类型之一：

- **文本**，用于自由填写的信息，比如艺术家或场馆。
- **数字**，用于数量和度量，比如期号或印刷数量。
- **日期**，用于任何与日历相关的信息，比如演唱会日期。
- **是 / 否**，用于简单的标记，比如“已签名”或“首版”。
- **单选**，用于你自定义的固定选项列表，比如出版社或等级。选项能让数据保持一致，因为所有人都从同一份列表中选择，而不是各自输入不同的写法。
- **评分**，用于一到五星的个人评分。

只要可能取值是一份已知列表，就优先选**单选**而不是**文本**。“Marvel”和“marvel comics”在你看来是一回事，但筛选功能可分辨不出来。

## 用字段分组让表单保持清晰

字段可以被组织到命名分组中，每个分组会在藏品表单上单独显示为一个区块。比如现成的漫画类型，就把字段分成了“出版信息”和“品相与分级”两组。未分组的字段会显示在最前面。

创建一个分组，起个名字，再把字段移进去。你可以调整分组内字段的顺序，也可以调整分组本身的顺序，让表单按照最适合你这个爱好的方式排列。

:::note
分组只影响藏品表单的呈现方式，不会改变数据本身，所以随时重新整理都没有问题。
:::

## 将类型关联到收藏

类型在被某个 @doc(collections.overview, "收藏") 启用之前不会起任何作用。创建或编辑收藏时，选择要启用哪些类型。一个收藏可以启用多个类型，同一个类型也可以用于多个收藏。启用后，该收藏中的藏品就可以选择这个类型并填写它的字段。
