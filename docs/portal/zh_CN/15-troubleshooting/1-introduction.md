# 故障排查与常见问题

当某个功能没有按你预期的方式运作时，从这里开始查找。这些页面直接给出答案，并在你需要完整说明时链接到详细文档。

1. @doc(troubleshooting.signIn)：忘记密码、丢失身份验证器、链接过期，以及打不开的邀请。
2. @doc(troubleshooting.emailDelivery)：为什么等待的邮件一直没收到，以及如何查看已发送的邮件记录。
3. @doc(troubleshooting.faq)：对大家都会问的问题给出简短答案，从"什么是副本"到"如何导出我的数据"。
4. @doc(troubleshooting.featureStatus)：一份如实列出当前已完整可用和仍在开发中功能的页面。

如果你的问题和某个具体功能有关，对应小节的索引通常能比这些通用页面更快地把你带到正确的操作说明页。
