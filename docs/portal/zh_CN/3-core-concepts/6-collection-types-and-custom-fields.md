# 收藏类型和自定义字段

漫画需要期号，葡萄酒需要年份，手表需要机芯信息。KolleK 不会替你猜测你在收藏什么，而是让你自己定义。这一页解释三个组成部分：类型、自定义字段，以及字段分组。

## 收藏类型

**收藏类型**描述你收藏的某一类东西：漫画、黑胶唱片、葡萄酒。它是容纳那些适用于这类东西的自定义字段的容器。

类型是账户级别、可以复用的。定义一次漫画类型，账户中的任何一个@doc(collections.overview, "收藏")都可以启用它。一个收藏可以同时启用多个类型，这适合混合型的收藏：Noah 的"音乐"收藏同时启用了黑胶唱片和 CD 两种类型，这样每个藏品都可以按其中一种来编目。

当一个藏品被赋予某个类型后，它的表单就会自动加上该类型定义的那些自定义字段。

## 自定义字段

**自定义字段**是某个类型要求填写的一项细节。每个字段都有自己的类型：

- **文本**，用于任意自由格式的内容，例如出版商或艺术家。
- **数字**，用于期号或发行年份。
- **日期**，用于封面日期。
- **是/否**，用于是否签名或是否首版。
- **下拉选择**，一个由你自定义选项的下拉菜单，例如评级 PSA 10、PSA 9，或未评级。
- **评分**，最多五星，用于你自己的"我的评分"。

字段的值是按藏品分别记录的。Emma 的"神奇蜘蛛侠 #1"期号为 1，出版商为 Marvel；她其他的漫画共用同样的字段，但各自有各自的值。

## 字段分组

当一个类型的字段很多时，**字段分组**可以让表单保持清晰易读。分组其实就是一个命名的小节：内置的漫画类型把它的字段分成了"出版信息"和"品相与评级"两组。冗长的表单因此变成了整洁的分区，而不是无止境的一长串。

## 内置类型

一个全新的账户自带十几种内置类型，让你不用从空白开始：漫画、集换卡、黑胶唱片、CD、DVD、硬币、邮票、图书、动作人偶/玩具、电子游戏、手表，以及葡萄酒，每一种都已经配好了合理的字段分组。你可以直接使用它们、调整它们，或者干脆不用它们，自己创建新的类型。

:::note
类型描述的是藏品，而不是副本。那些因你所拥有的每个实物而不同的字段，比如品相或编号，应该放在副本上。参见@doc(items.itemsVsCopies)。
:::

## 接下来可以做什么

- 在@doc(collectionTypes.setup)中一步步创建或调整一个类型。
- 在@doc(collectionTypes.importExport)中把一个类型定义分享给其他人。
- 在@doc(items.addAndEdit)中看看字段在实际中的效果。
