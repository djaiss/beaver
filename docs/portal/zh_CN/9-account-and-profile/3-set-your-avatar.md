# 设置你的头像

头像是代表你的小图片，会出现在 KolleK 的各个地方：成员列表、动态记录中你的条目旁边，以及用户菜单里。如果你没有设置头像，KolleK 会显示一个自动生成的占位图代替。本页介绍如何上传头像，以及如何再次移除它。

## 上传头像

从 @doc(profile.edit, "你的个人资料") 中选择更换头像，选取一张图片文件。

几条实用规则：

- 使用 **JPEG**、**PNG** 或 **WebP** 格式的图片。
- 文件最大可以到 **5 MB**。
- 图片会以正方形显示，所以一张大致方形的图片效果最好，过宽或过高的图片会被裁切。

保存之后，你的新头像会立刻出现在各处。

## 移除你的头像

在同一个地方选择移除头像，KolleK 会重新为你显示自动生成的占位图，你可以随时再上传一张新图片。

## 接下来

- 在 @doc(profile.edit) 中更新你的姓名或昵称。
- 在 @doc(activity.feedAndAuditTrail, "动态记录") 中看看你的姓名和头像会出现在哪些地方。
