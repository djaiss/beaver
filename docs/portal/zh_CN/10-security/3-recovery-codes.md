# 保存和使用恢复代码

如果你丢失了认证器，恢复代码就是你回到账户的方式。开启 @doc(security.twoFactorAuth, "两步验证") 时，KolleK 会生成八个恢复代码。每个代码只能使用一次，用来代替应用里的验证码。

手机会丢、会摔坏、会被换掉。恢复代码就是这种普通的倒霉日子和被锁在自己的目录之外之间的那道屏障。

## 你在哪里能拿到它们

代码会在你完成两步验证设置确认后立即显示，那一刻就是你该保存它们的时候。

保存它们的好去处：

- 密码管理器，放在你 KolleK 条目的备注里。
- 家里抽屉中打印出来的一张纸。
- 一份你会备份的加密文件。

不适合的保存方式是只放在手机里，因为你需要用到它们的场景，恰恰就是手机不在身边的场景。

:::warning
如果你同时丢失了认证器和恢复代码，就无法完成两步验证这一步，可能会被锁在自己的用户之外。这没有自助解决的办法，所以现在就把代码存到安全的地方。
:::

## 用一个代码登录

当 KolleK 要求你输入六位数认证码，而你没办法提供时：

1. 在两步验证的确认界面，输入你的一个恢复代码，代替应用里的验证码。
2. 你会像平常一样登录成功。

就这么简单。这个验证环节既接受当前的认证码，也接受一个尚未使用过的恢复代码。

## 每个代码只能用一次

一个恢复代码在你使用的那一刻就会被消耗掉，之后再也无法使用，你剩下的代码依然有效。请在你保存代码的地方把用过的划掉。

:::note
如果你的代码所剩不多，或者怀疑有人看到过它们，可以关闭两步验证再重新开启。重新开启会生成一套全新的八个代码，并让旧代码全部失效。
:::

## 重新登录之后

如果你是因为彻底丢失了认证器才使用恢复代码，花两分钟把设置理顺：在安全设置中关闭两步验证，再用你的新设备重新开启它。你会得到一个新的二维码可以扫描，以及一套全新的恢复代码需要保存。
