# 用两步验证保护你的账户

两步验证会在登录时增加一个步骤。你的密码通过验证后，KolleK 还会要求你输入手机上认证器应用生成的六位数验证码。即使有人知道了你的密码，没有这个验证码也无法登录。

这是 KolleK 提供的最有效的安全控制手段，设置起来只需要几分钟。

## 你需要准备什么

手机上的一个认证器应用，任何支持基于时间的一次性验证码的应用都可以。如果你曾经扫过二维码来保护另一个账户，那你已经有一个可用的应用了。

## 开启它

::::steps
:::step title="打开你的安全设置"
进入个人资料，打开安全区域，选择设置**两步验证**。
:::

:::step title="扫描二维码"
KolleK 会显示一个二维码。打开你的认证器应用，添加一个新账户，扫描这个码。应用会开始为 KolleK 显示一个六位数验证码，每 30 秒变化一次。

::screenshot{label="带二维码的两步验证设置界面"}
:::

:::step title="用一个验证码确认"
把应用上当前的六位数验证码输入到确认框中并提交。这一步用来确认应用和 KolleK 已经同步，之后你的登录方式才会真正改变。
:::

:::step title="保存你的恢复代码"
KolleK 会生成八个恢复代码。把它们复制到手机以外的安全地方，比如密码管理器或打印出来的一张纸。每个代码在你日后丢失认证器时，都可以让你登录一次。

::screenshot{label="设置完成后显示的八个恢复代码"}
:::
::::

:::warning
如果你丢失了认证器，又没有保存恢复代码，就无法完成两步验证这一步，可能会被锁在自己的用户之外。请在关闭这个页面之前先保存好这些代码。
:::

## 登录时会有什么变化

从现在起，用邮箱和密码登录会多出一个步骤。密码通过验证后，KolleK 会要求你输入认证器应用上当前的验证码，输入后即可登录。

如果你无法使用认证器应用，可以改用你的一个 @doc(security.recoveryCodes, "恢复代码")。

:::note
用 @doc(auth.magicLinks, "魔法链接") 登录时不会要求两步验证码。能访问你的邮箱本身已经充当了第二重验证，所以请相应地保护好那个邮箱。
:::

## 关闭它

你可以在同一个安全区域关闭两步验证。这样做会从登录流程中移除验证码这一步，同时也会删除你的恢复代码，以及和认证器应用的配对关系。如果之后再次开启，你需要重新扫描一个新的二维码，并获得一套全新的恢复代码。
