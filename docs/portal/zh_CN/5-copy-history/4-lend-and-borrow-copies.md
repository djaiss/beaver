# 借出和借入副本

副本迟早会离开你的手中：朋友借走了它，展馆展出了它，或者你自己借入了别人的东西。借还记录追踪的正是这种保管权的变动，让你随时知道一个副本在哪里、在谁手上，以及它离开时是什么状况。

核心思路是：借出转移的是**保管权**，而不是**所有权**。你借出去的副本仍然是你的，仍然属于你的收藏，仍然算在你所持有的东西里。借还记录解释的正是它为什么不在你的架子上。

## 记录一次借出

Priya 把她那块 1968 年的欧米茄超霸表借给当地一场钟表展览，为期三个月。以下是她记录的过程。

::::steps
:::step title="打开副本的历史记录"
打开藏品，进入它的**历史**标签页，选择该副本，打开**借还**部分。

::screenshot{label="展开了借还部分的历史标签页"}
:::

:::step title="选择方向"
如果副本正在离开你，选择**借出**；如果是别人的副本进入你的保管，选择**借入**。
:::

:::step title="记录对象、原因和时间"
填写**对象**（借出方或借入方的人或机构）、**目的**（例如"市立博物馆的钟表展览"）、**借出日期**，以及约定好的**归还日期**（如果有）。
:::

:::step title="记录离开时的品相"
设置**离开时品相**，也就是副本离开那一刻的@doc(conditions.overview, "品相")。这是你日后发现损坏的对照基准。你还可以选填**押金**金额（如果收取了的话），并选择是否要把这次借出**纳入来源记录**。保存这条借还记录。
:::
::::

在借出期间，这个副本在任何地方出现时都会显示为**已借出**。它仍然算作持有中，因为变动的是保管权，而不是所有权。

## 借还状态

一条借还记录始终处于以下六种状态之一：

- **已计划**：已经约定好，但副本还没有离开。
- **进行中**：副本目前在外（借入的情况则是目前在你手上）。
- **已逾期**：归还日期已过，但还没有归还。
- **已归还**：已经回来，归还情况已记录。
- **已取消**：在发生之前就取消了。
- **已丢失**：副本没能回来。

:::note
KolleK 会为你标出逾期的借还。每天运行的后台任务会把超过归还日期仍处于进行中的借还标记为已逾期，这样没有归还的副本会自动浮现出来，而不用全靠你自己记着。
:::

## 归还一次借出

归还是单独的一步，也是借还记录真正发挥作用的地方。

::::steps
:::step title="打开借还记录"
从副本的**借还**部分，打开这条进行中的借还记录。
:::

:::step title="记录归还情况"
选择归还，并设置**归还日期**。
:::

:::step title="记录归还时的品相"
设置**归还时品相**。如果它和离开时的品相不同，你就记录下了运输或使用过程中造成的损坏，而且带有明确的日期和对方信息。这里记录的品相会成为副本当前的品相。
:::
::::

当超霸表从展览归还时，Priya 记录了归还日期，并确认它的品相和离开时一样。如果展览过程中表壳被划伤了，这条借还记录上前后的品相对比就会准确说明这是什么时候、在谁的保管下发生的。

:::warning
在副本离开之前，如实记录它离开时的品相。如果你跳过这一步，就失去了对照基准，日后一旦对损坏产生争议，就变成了各说各话。
:::

## 借入

**借入**方向的操作方式与借出完全相同，只是方向相反：你记录这是谁的副本、你为什么持有它，以及什么时候要归还。借入的副本会被单独追踪，这样那些你只是暂时保管、并不属于你的东西，就不会不知不觉混入你自己的收藏。

## 接下来可以做什么

- 在@doc(conditions.overview)中了解借出和归还时使用的品相等级。
- 一条被标记为纳入来源的借还记录会出现在所有权故事里。参见@doc(copies.traceProvenance)。
- 在@doc(copyHistory.readTimeline)中查看借还记录和其他所有记录一起呈现的样子。
