# 核心功能

你已经了解了这些概念，现在该动手了。本节将带你走过搭建和浏览目录的日常任务：创建收藏、添加藏品和副本、上传照片、打标签，以及关注谁改动了什么。

每一页都假设你已经读过 @doc(coreConcepts.index) 中对应的概念页面，因此只会回链过去，而不会重复讲解。

请按以下顺序阅读：

1. @doc(collections.manage)。搭建承载其他一切内容的容器。
2. @doc(collections.chooseView)。网格、列表或表格，以及收藏内搜索的行为方式。
3. @doc(items.addAndEdit)。核心任务：把东西放进目录里。
4. @doc(copies.track)。记录每一件实体副本，它的品相、位置和状态。
5. @doc(items.addPhotos)。封面和其他照片。
6. @doc(items.tagAndFind)。跨收藏使用的灵活标签。
7. @doc(activity.feedAndAuditTrail)。查看谁在什么时候改动了什么。

如果你还没有编目过任何东西，@doc(gettingStarted.quickStart, "五分钟快速上手") 是最快走完这些任务的第一遍。
