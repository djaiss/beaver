# 为藏品打标签并查找

@doc(tags.overview) 是标记藏品共同特征最轻量的方式，可以跨越你账户里的每一个收藏。本页介绍如何添加标签、移除标签，以及一些能让标签保持实用的习惯。

## 给藏品添加标签

在藏品表单上，使用**标签**字段：

- **选择账户共享列表中已有的标签**，或者
- **输入一个新标签**，它会当场被创建，之后随处可以复用。

一件藏品可以带任意多个标签。Emma 给她的签名版漫画都打上“已签名”标签，无论它们放在她的漫画收藏还是图像小说收藏里，这个标签在两边都是同一个标签。

## 从藏品上移除一个标签

编辑这件藏品，从标签字段中移除这个标签。标签本身仍然保留在账户列表中，其他仍带有这个标签的藏品不受影响。

## 维护共享列表

因为标签是账户级别共享的，标签列表本身要在一个地方统一管理，也就是账户设置。在那里重命名和删除标签的方法见 @doc(tags.manageAccount)。

:::note
从账户中删除一个标签，会把这个标签从所有带有它的藏品上移除。藏品本身不会受到其他任何影响。
:::

## 保持标签实用的习惯

- **给特征打标签，而不是给类别打标签。**“已签名”“首印”“需要修复”“待出售”都是好标签。“漫画”是一个收藏，“Marvel”多半应该是一个 @doc(categories.organizeItems, "分类")。
- **先复用，再新建。** 同一个意思的两种写法（“已签名”和“Signed”）会把你的目录一分为二，如果已有标签合适，就直接选用它。
- **保持列表简短。** 一份你真正会用来筛选的几个标签，胜过五十个你要费劲才记得住的标签。
