# 添加和编辑藏品

这是你会用得最多的一页：把条目放进你的目录里。本页会逐个字段讲解藏品表单，说明哪些部分是选填的（几乎全部都是），并介绍如何编辑和删除。

如果你还分不清藏品和副本的区别，先阅读 @doc(items.itemsVsCopies)。简单来说：藏品描述的是这一类东西，而副本记录的是你实际拥有的实物。

## 谁可以这样做

添加和编辑藏品需要**编辑者**或**所有者** @doc(accounts.usersAndRoles, "角色")。

## 添加一件藏品

::::steps
:::step title="打开收藏"
打开这件藏品所属的收藏，选择**新建藏品**。

::screenshot{label="收藏视图，新建藏品按钮"}
:::

:::step title="给它命名"
输入**名称**，这是唯一必填的字段。Emma 输入了“神奇蜘蛛侠 #300”。其余内容现在填或以后再填都可以。
:::

:::step title="分类它"
可以选填**类型**、**分类**、**套装**和**系列**，并添加**标签**。类型是其中最重要的一项：选定类型后，该类型的自定义字段就会出现在表单上。

::screenshot{label="藏品表单，类型和分类字段"}
:::

:::step title="填写详情"
在同一份表单上，填写类型提供的**自定义字段**，上传**照片**，并记录你拥有的**副本**。
:::

:::step title="保存"
保存这件藏品，它会立即出现在收藏里。
:::
::::

## 表单逐字段详解

- **名称。** 必填，也是唯一必填的项。
- **描述。** 自由文本，用来记录其他字段放不下的内容。
- **类型。** 这件藏品属于哪个 @doc(collectionTypes.overview, "收藏类型")，只会提供该收藏已启用的类型。类型决定了下方会出现哪些自定义字段。
- **分类。** 这件藏品在这个收藏内归档在哪里，见 @doc(categories.organizeItems)。
- **套装。** 你正在凑齐的一份有限清单，见 @doc(sets.trackCompletion)。
- **系列。** 一个可以跨收藏的系列作品，见 @doc(series.groupFranchise)。
- **标签。** 选择已有的 @doc(tags.overview, "标签")，或输入一个新标签，它会当场被创建。
- **自定义字段。** 由所选类型定义的内容：文本、数字、日期、是 / 否开关、单选列表，以及最多五星的评分。字段会按照该类型设定的分组方式显示。
- **照片。** 详见 @doc(items.addPhotos)。
- **副本。** 一件或多件实体副本，直接在同一份表单里添加，详见 @doc(copies.track)。

不必强求一次性填完所有内容。现在填个名字、之后再补充细节是完全可行的工作方式，用的还是同一份表单。

## 编辑一件藏品

打开这件藏品并选择编辑它。表单还是同一份，只是已经预先填好了内容，改动你需要的部分后保存即可。

## 删除一件藏品

打开这件藏品，选择删除，然后确认。

:::warning
删除一件藏品会把它和它的副本一起送入回收站，在保留期（默认 30 天）结束后会被永久移除。
:::

在此之前，你都可以把它找回来，见 @doc(dataSafety.restoreFromTrash)。
