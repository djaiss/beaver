# 为藏品添加照片

照片是让一份目录一眼就能认出来的关键。本页介绍如何为一件藏品上传照片、排列顺序，以及选择封面。

照片挂在**藏品**上，而不是某一个具体的副本上。它们描述的是这件东西看起来是什么样子，这对每一个副本来说都是一样的。和某个特定副本相关的文件（收据、证书、只针对某一个副本的品相照片）应该放进那个副本的 @doc(copies.attachDocuments, "文档") 里。

## 上传照片

在添加或编辑藏品时，使用照片区域上传一张或多张图片。

- **格式。** JPEG、PNG、WebP 和 GIF。
- **大小。** 每张照片最多 10 MB。

Emma 为她的《神奇蜘蛛侠》#300 上传了两张照片：封面和封底。

## 选择主图

当一件藏品有多张照片时，其中一张会作为**主图**，也就是显示在网格和列表中的封面。默认由第一张照片担任这个角色，你可以随时换成另一张。你还可以重新排列照片，让附加照片按你想要的顺序显示。

## 移除一张照片

你可以随时从藏品中删除一张照片。

:::note
删除照片会立即移除该照片及其文件。照片不会进入回收站，删除后无法恢复。
:::

## 批量管理照片

你整个账户里的每一张照片都可以在一个界面里被搜索、筛选和管理，也就是照片库，这也是唯一提供批量操作的地方，见 @doc(photos.library)。

## 接下来

- 记录照片背后的实体副本：@doc(copies.track)。
- 一次性处理你所有的图片：@doc(photos.library)。
- 把和副本相关的文件放在它该在的地方：@doc(copies.attachDocuments)。
