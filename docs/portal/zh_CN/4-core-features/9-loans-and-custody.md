# 借出与保管

借出是保管权的临时转移，而所有权并不发生任何转移。当你把一件藏品借给朋友、画廊或博物馆时，它仍然归你所有。当你借入一件藏品时，它仍然属于别人。**借出**板块是整个账户范围内的总览，展示当前不在你手中、或以借入方式在你手中的一切藏品。

查看该板块对任何角色开放。记录、归还、编辑和删除一笔借出，需要**编辑者**或**所有者**角色。

## 两个方向

每一笔借出都指向两个方向之一，板块一次只显示一个方向。使用顶部的切换按钮在两者之间切换。

- **借出。** 属于你、但由别人持有的一件藏品。只要一笔向外借出处于进行中或已逾期，它的副本在你的收藏中就显示为**已借出**，因为它实际上并不在你身边。
- **借入。** 属于别人、目前由你持有的一件藏品。借入的藏品绝不会改变你自己副本的显示方式，因为它从来都不归你所有。

## 各标签页展示的内容

在同一个方向内，各标签页以不同的方式对同一批借出进行切分。

- **全部借出。** 该方向下的每一笔借出，配有搜索框，以及按收藏、状态和排序方式的筛选。
- **到期与逾期。** 三个列表：已过到期日的借出、将在三十天内到期的借出，以及根本没有到期日的无期限借出。
- **风险与例外。** 需要再看一眼的借出：逾期、丢失、归还时品相变差、缺少到期日、借出时未记录品相，或借出时档案中没有任何文件。
- **按对方。** 每个人或机构一张卡片，让你一次性看清某一位借入方或出借方手上持有的全部藏品。
- **押金。** 在所有进行中的借出里，你所持有或别人欠你的押金，以及附带押金的那些借出。
- **时间线。** 即将到期的日期、近期归还的藏品，以及近期借出的藏品。

顶部的统计磁贴是快捷方式：每一个都会打开回答它的那个标签页。

## 记录一笔借出

你可以直接在该板块内发起一笔借出，无需先去查找副本。

::::steps
:::step title="打开新建借出抽屉"
选择**新建借出**。先选定方向，然后从收藏逐层向下走到藏品，再到正在移动的那一个确切副本。
:::

:::step title="填写对方和日期"
填写这件藏品要交给谁或来自谁、它离开的日期，以及一个到期日。若没有约定归还日期，请勾选**无期限**。
:::

:::step title="记录品相和任何押金"
选择**借出时品相**，以便日后归还时可以与之对照；若有押金易手，请记录**押金**。押金币种默认采用该收藏的币种。
:::

:::step title="若属于来历的一部分，请标记为来历"
对机构借展或展览，勾选**计入来历**，系统会生成一条对应的来历事件。对于非正式的个人借出，则不勾选，它只保留在借出历史中。
:::
::::

### 每个副本同时只能有一笔进行中的借出

一个实体副本一次只能在一个地方，因此一个副本最多只能有一笔**进行中的向外借出**。如果你试图借出一个已经在外的副本，板块会予以拦截，并要求你先归还当前这笔借出。此规则在 JSON API 中同样适用。

## 归还一笔借出

结束一笔借出是一个独立的步骤，而不是一次编辑，这样它才能记下编辑无法记下的信息。

::::steps
:::step title="打开借出并标记为已归还"
从任意列表中打开这笔借出，然后选择**标记为已归还**。
:::

:::step title="记录归还"
填写藏品归还的日期以及**归还时品相**。设定归还时品相会更新副本的当前品相，并把副本重新纳入你的保管之下。
:::
::::

当归还时品相比借出时品相更差时，这笔借出会被标记为可能受损，既标记在借出本身上，也出现在**归还时变差**风险列表中。

## 导出仍在外的藏品

**导出仍在外的藏品**按钮会下载当前方向下所有进行中借出的 CSV 文件，让你手边有一份清单，清楚列出当前在别人手中、或在你手中的藏品。
