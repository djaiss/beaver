# 创建和管理收藏

@doc(collections.overview, "收藏") 是承载其他一切内容的容器，因此它通常是你要创建的第一样东西。本页介绍如何创建一个收藏、表单上的每一个选项、之后如何编辑，以及删除时到底会发生什么。

## 谁可以这样做

创建、编辑和删除收藏需要**编辑者**或**所有者** @doc(accounts.usersAndRoles, "角色")。查看者可以浏览收藏，但不能创建或修改它们。

## 创建一个收藏

Noah 正打算为他的黑胶唱片建一个目录，以下是他的操作。

::::steps
:::step title="新建一个收藏"
在收藏列表页面，选择**新建收藏**。

::screenshot{label="收藏列表页面，新建收藏按钮"}
:::

:::step title="命名并描述它"
给它起一个**名称**，比如“黑胶唱片”，可以选填一段简短的**描述**和一个**表情符号**，让它在列表中更醒目。
:::

:::step title="选择它的收藏类型"
选择要启用哪些 @doc(collectionTypes.overview, "收藏类型")。Noah 选择了现成的**黑胶唱片**类型，这样他的藏品就会有艺术家、专辑和压制版本等字段。你可以启用多个类型，也可以一个都不启用，之后随时可以更改。
:::

:::step title="设置货币和可见范围"
为这个收藏中的价值选择**货币**，并设置它的**可见范围**。如果拿不定主意，保留默认值就好，私密是最安全的起点。

::screenshot{label="收藏表单，货币和可见范围字段"}
:::

:::step title="保存"
保存这个收藏。它会出现在你的列表中，空空如也，等待第一件藏品的到来。
:::
::::

## 每个字段详解

- **名称。** 收藏在各处显示的名字，必填。
- **描述。** 一句话说明它收藏的是什么，选填，但当你的收藏越来越多时会很有帮助。
- **表情符号。** 从固定的十二个图案中选一个作为视觉标记（📦 📚 💿 🃏 🍷 🎮 🧸 🪙 🖼️ ⌚ 👟 📷），选填。
- **收藏类型。** 你启用的类型决定了这个收藏里的藏品可以记录哪些自定义字段。你可以启用不止一个类型，例如把漫画和图书都启用在同一个“阅读”收藏里。
- **货币。** 这个收藏里所有的金额（价值、统计数据）都使用这种货币，共有十八种货币可选。它可以和账户的默认货币不同，如果你用欧元买红酒，其他东西却用美元结算，这个设置就很方便。
- **可见范围。** 这个收藏面向谁：**私密**（只有你自己）、**共享**（账户内的所有人）或**公开**（拥有链接的任何人，只读）。这个设置目前会被记录下来，等分享功能上线后才会真正生效。@doc(sharing.overview, "可见范围与分享") 这篇概念页面解释了这套模型和它目前的进展，@doc(collections.share) 则介绍了具体如何修改它。

## 编辑一个收藏

打开这个收藏并选择编辑它。会出现同一份表单，同样的字段，你可以随时修改其中任何一项。重命名收藏或换一个表情符号都不会影响它内部的任何内容。

## 删除一个收藏

打开这个收藏，选择删除，然后确认。

:::warning
删除一个收藏也会把它里面的每一件藏品都送入回收站。收藏及其藏品会在回收站中保留一段有限的时间（默认 30 天），之后会被永久移除。
:::

在它还留在回收站期间，你仍然可以改变主意，见 @doc(dataSafety.restoreFromTrash)。
