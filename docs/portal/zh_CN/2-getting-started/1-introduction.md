# 入门指南

欢迎使用 KolleK。本节将带你从"刚接触这款产品"逐步过渡到完成第一件藏品的编目，过程中不需要你提前掌握任何进阶知识。

请按以下顺序阅读各页面：

1. @doc(kollek.whatIs)。了解这款产品能做什么，以及是否适合你。
2. @doc(kollek.hostingOptions)。了解你将如何运行它，以及为什么应用内不需要支付任何费用。
3. @doc(accounts.create)。注册并获得属于你的工作空间。
4. @doc(auth.signIn)。了解以后重新登录的各种方式。
5. @doc(gettingStarted.checklist)。值得优先完成的五件事。
6. @doc(gettingStarted.quickStart)。从头到尾完成第一件藏品的编目。

如果你已经了解 KolleK 是什么，只想尽快上手，可以直接跳转到 @doc(accounts.create)，或前往 @doc(gettingStarted.quickStart, "快速上手")。
