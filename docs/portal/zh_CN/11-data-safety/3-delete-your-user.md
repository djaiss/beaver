# 删除你的用户

删除你的用户会把你这个人从 KolleK 中移除。这与删除账户不是一回事：账户是共享的工作空间，@doc(accounts.delete, "删除账户")会摧毁所有人的一切内容。本页只涉及删除你自己。

## 决定之前

有两种情况看起来像是"删除我的用户"，但其实并不是：

- **你想让一切都消失。** 如果你是所有者，并且想彻底移除整个目录和工作空间，你需要的是 @doc(accounts.delete)。
- **你只是想退出一个共享账户。** 删除你的用户只会移除你自己，账户和其中的目录会留给其他成员。

如果你是账户唯一的所有者，且账户中还有其他成员，请先通过@doc(collaboration.manageMembersAndRoles, "成员管理")把所有者身份提升给别人，以免账户失去所有者。

## 删除你自己

::::steps
:::step title="打开个人资料设置"
进入个人资料设置，找到页面底部的危险操作区域。
:::

:::step title="说明离开原因"
需要填写一个原因（几个字即可，至少三个字符）。这条信息会发送给实例运营者，帮助改进 KolleK。

::screenshot{label="删除用户表单，含原因输入框"}
:::

:::step title="确认"
在弹出的对话框中确认删除。你会立即被登出，此后你的登录信息将无法再使用。
:::
::::

:::warning
删除你的用户是永久性操作。你的登录信息会消失，且无法在应用中恢复。你的邮箱地址会重新变为可用，因此你之后可以用它注册一个全新的账户，但那将是一个空账户。
:::

## 你留下的痕迹会怎样

账户的活动历史会保持完整：你创建的记录会保留你当时的姓名，因此当你离开后，共享工作的审计记录不会出现缺失。

## 接下来去哪里

- 更倾向于自动清理？参见 @doc(users.inactiveDeletion)。
- 从共享账户中移除其他人，请参见 @doc(collaboration.manageMembersAndRoles)。
