# Statistiques

Une fois que vous avez suffisamment catalogué, KolleK commence à vous le rendre : ce que vaut votre collection, comment elle a évolué et où se situe la valeur. Cette courte section explique les écrans qui vous montrent tout cela.

Lisez les pages dans cet ordre.

1. @doc(insights.collectionStatistics). Chaque chiffre et chaque graphique de l'écran de statistiques d'une collection, et d'où provient chaque donnée.
2. @doc(insights.dashboard). L'écran d'accueil après votre connexion.

Les statistiques ne sont fiables que si les données derrière le sont. Si les chiffres semblent maigres, la solution consiste généralement à enregistrer des @doc(copies.recordPaymentsAndValue, "évaluations et transactions") pour vos exemplaires.
