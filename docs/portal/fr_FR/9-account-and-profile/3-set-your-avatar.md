# Définir votre avatar

Un avatar est la petite image qui vous représente dans KolleK : dans la liste des membres, à côté de vos entrées dans le fil d'activité, et dans le menu utilisateur. Sans avatar, KolleK affiche à la place un espace réservé généré automatiquement. Cette page couvre l'ajout d'un avatar et sa suppression.

## Ajouter un avatar

Depuis @doc(profile.edit, "votre profil"), choisissez de modifier votre avatar et sélectionnez un fichier image.

Quelques règles pratiques :

- Utilisez une image **JPEG**, **PNG** ou **WebP**.
- Le fichier peut faire jusqu'à **5 Mo**.
- L'image est affichée en carré, donc une image à peu près carrée rendra le mieux. Une image large ou haute est recadrée.

Enregistrez, et votre nouvel avatar apparaît partout immédiatement.

## Retirer votre avatar

Depuis le même endroit, choisissez de retirer l'avatar. KolleK revient à l'affichage de l'espace réservé généré pour vous. Vous pouvez ajouter une nouvelle image à tout moment.

## Et ensuite

- Mettez à jour votre nom ou votre surnom dans @doc(profile.edit).
- Découvrez où apparaissent votre nom et votre image dans @doc(activity.feedAndAuditTrail, "le fil d'activité").
