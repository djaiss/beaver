# Développeurs et API

KolleK propose une API JSON complète qui reflète l'application web : tout ce que vous pouvez faire dans l'application, vous pouvez le faire avec un jeton et un client HTTP. Cette section vous permet de vous authentifier et de vous orienter. La référence complète des points d'accès est générée à partir du code et toujours à jour, donc ces pages ne la répètent volontairement pas.

Lisez les pages dans cet ordre :

1. @doc(api.overview). Ce que couvre l'API, comment elle est structurée, et où se trouve la référence générée.
2. @doc(api.authenticate). Obtenez un jeton et effectuez votre première requête.
3. @doc(api.rateLimitsAndConventions). Pagination, montants, limitation de débit et comportement des erreurs.
4. @doc(webhooks.overview). Enregistrez des points d'accès dès aujourd'hui, et comprenez précisément ce qui se déclenche (et ce qui ne se déclenche pas encore).

Pour le détail de chaque point d'accès (paramètres, formes de réponse, exemples), utilisez la référence générée sur `/docs/api` de votre instance.
