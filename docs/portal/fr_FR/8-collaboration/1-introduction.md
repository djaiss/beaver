# Collaboration

Un compte KolleK est un espace de travail partagé. Vous pouvez le garder pour vous seul, ou y faire entrer votre foyer, votre club, ou vos associés, chacun avec le niveau d'accès approprié. Cette section explique comment faire entrer des personnes, décider de ce qu'elles peuvent faire, et partager une collection avec l'extérieur.

Lisez les pages dans cet ordre.

1. @doc(collaboration.invitePeople). Envoyez une invitation et découvrez ce que la personne invitée va vivre.
2. @doc(collaboration.manageMembersAndRoles). Changez ce que les membres existants peuvent faire, ou retirez les.
3. @doc(collaboration.rolesInPractice). Une référence rapide de qui peut faire quoi.
4. @doc(collections.share). Rendez une collection visible pour votre compte, ou pour toute personne disposant d'un lien.

Si vous n'avez pas encore lu la page de concept sur @doc(accounts.usersAndRoles, "les comptes, utilisateurs et rôles"), commencez par là. Elle explique le modèle sur lequel ces pages s'appuient.
